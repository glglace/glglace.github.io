20070404--嘰哩呱啦ACE--第133回--檔案急救真驚險
  • 越急的時候，這些科技的玩意兒似乎就越不聽話！
  • 昨天在趕著寫一份會議討論提案的時候，已經寫到快要完成的時候，很開心的按下「存檔」，竟然發生儲存錯誤，還好文書處理軟體還沒當掉，用「另存新檔」存一份到桌面上，然後再繼續工作，最後將檔案「剪下」、「貼上」到行動碟。
    　
  • 後來又想到要改幾個字，從行動碟叫出來改，要存檔時....完蛋！檔案竟然就不見了！我覺得大概是那條USB延長線有故障或鬆脫，但，ㄧ切都不見了！三個小時的工夫難道就這麼毀了嗎？非常不甘心！
    　
  • 我想起昨天晚上看download網路情報誌的2007年2月號裡面提到有ㄧ個記憶卡救援程式 Smart Data Recovery，趕緊去下載來救看看，結果！有救回來，但是檔案內容全都變成亂碼！真是令人傷心...
    　
  • 要我重寫？想到就沒力 ....
    　
  • 趕快再從我中看不中用的大頭中，挖掘過往的記憶，想起以前曾經有ㄧ次相機的記憶卡有問題的時候，用了某ㄧ個軟體有成功救回，但想不起軟體的名稱...
    　
  • 真的要放棄嗎？再想想，有ㄧ個TOGET網站，將許多好用軟體分門別類收集，可以去其分類全覽中，碰碰運氣，我選了檔案工具類，然後選檔案急救子類，軟體排序選擇「免費軟體」，然後看到 PC INSPECTOR File Recovery就想起來是這個！
    　
  • 這又是一個來自德國的免費軟體！下載安裝好之後，救救看，萬歲！成功救回！
    • 要下載軟體最好到該軟體的官方網站，比較不會被植入病毒或木馬，下載到的版本也會是最新的。
    • 在PC INSPECTOR File Recovery的下載網頁中， 有ㄧ段他們關於「免費軟體」的解釋我覺得寫得很不錯，但並非每個人對「免費軟體」的解釋均是如此，會有授權範圍的差別，翻譯給大家看看：
      1. 免費軟體是指可以免費使用的軟體。但作者仍保留該軟體的在著作權法上的所有權利。
      2. 所謂的「免費」，是指您可以免費取得、使用、和散佈本軟體。
