20091015--嘰哩呱啦ACE--第634回--買記憶卡和讀卡機
  • 今天下午中華電信來家裡施工，把原本的 ADSL 數據機換掉，就升級成光世代了，只花了十分鐘，真是快呀！到中華電信網路連線測速區測試看看，下載速度是： 8.08Mbps！
    • 由於我自己個人使用習慣不佳，行動碟沒有一隻的蓋子是還在的，老是亂丟亂摔，目前已經壞掉了三隻。有了多次慘痛的檔案消失的經驗，這次我購買行動碟，我選擇了買記憶卡和讀卡機的組合，因為我想，如果摔到地上了，頂多是讀卡機壞掉，記憶卡應該還可以用，不過我不知道這樣的想法能不能成立？過一陣子有摔過再跟大家報告。


    • 10/10 先載老婆去埔基，然後帶女兒去埔里自然牙醫，看完牙醫後，老婆還沒輪到看診，所以我就帶小孩們先去逛埔里的順發3C量販。

    • 我買了 PHILIPS 的 SDHC讀卡機 $129元，我選透明的，因為如果有多張卡片抽換的時候，比較能一眼看到裡面的是哪張記憶卡。

    • 還有SanDisk 的 8G記憶卡 $599元，除了想測試看看摔到的時候是不是比較不會壞，此外就是記憶卡通常都有一個防寫卡榫，出門在外使用陌生的電腦時，可以自我保護，百毒不侵。

    • 買完離開店家後，去 7-11 買吃的，小孩在車上吃，我則打開記憶卡來把玩，發現怎麼沒有防寫卡榫？趕快回順發問店員換一個，還有沒有回到家才檢查，那我一定會很火。

    • 我並不常購買 3C產品啦 (您相信嗎？哈！)，我是說跟買書比起來，這次想到說順發有會員迴轉金，問一下我戶頭有多少，店員說有161元，但滿百元才能抵用，所以我付了 129+599-100=628元。不知道會不會比在網路上買的貴？

    • 回家後上網查一下價錢，看看自己是不是買貴了？可卻但到同樣都是 SanDisk 的 8G記憶卡，卻有好幾種價錢？
      • SanDisk 8GB SDHC 記憶卡(公司貨)  $748
      • SanDisk Ultra SDHC 8GB記憶卡 $980
      • SanDisk Extreme SDHC 8GB記憶卡 $2480

    • 研究了一下，發現價差是因為記憶卡的讀寫速度，去查了一下維基百科SDHC條目，以下為引用文字：
      • SDHC，全名「Secure Digital High Capacity」，是SD卡協會（SD Card Association）在2006年3月發表SD 2.0規格的Secure Digital（以下簡稱「SD」）高容量版本。
      • SDHC與SD的主要差異在於，舊版本用FAT16檔案系統，意思是管理檔案所在位置的表格用16位元表 示，所以最多隻能管理65536個範圍，再考慮每個範圍能儲存32KB的資料量，所以65536 × 32KB = 2GB，SD卡容量上限只能到達2GB。為解決FAT16格式可支援容量有限的問題，SDHC改用了FAT32格式；依規格定義，容量最大可達到 32GB。
      • SDHC記憶卡分為不同的級數（Class），表示該記憶卡能夠達到的最快的寫入速度，分為三種寫入速度：Class 2、Class 4及Class 6。SD卡協會有強制規定，所有符合SDHC規範的裝置都必須標明「SDHC」標誌。
        • Class 2: 寫入速度至少 2 MB/s以上
        • Class 4: 寫入速度至少 4 MB/s以上
        • Class 6: 寫入速度至少 6 MB/s以上

    • 於是仔細看了以下我買的記憶卡，果然有一個 Class 2 的標誌，想起以前曾看過的一篇部落格文章，狗狗的攝影日記PhotoFast 16G SDHC，裡面有一個記憶卡測速軟體，ATTO Disk Benchmark，下載來玩玩看！


    • 執行該軟體之後，首先傷腦筋的就是一些選項要如何設定，根據我多次測試，不管設定什麼好像測試結果都差不多，但考慮到測試所花的時間，建議您設定：
      • Transfer Size 0.5KB - 1024.0 KB
      • Transfer Length 32MB
      • Overlapped I/O 打勾

    • 按下 Start 按鈕後，等個一兩分鐘就會看到測試結果，一搬來說讀取速度都是大於寫入速度的，所以速度等級主要是觀察寫入速度，以我買的這張SanDisk SDHC 8GB 記憶卡為例，標示 class 2，但實測發現有 6 MB/s 以上的水準，算是合格了。


    • 測出興趣來了，所以把我的另外兩隻常用的行動碟也拿來測測看，左上圖為SanDisk 2G 行動碟，右上圖為 SanDisk 4G 行動碟，發現寫入速度都比新買的SDHC 8GB 記憶卡好一點點，但讀取速度則分別快上兩、三倍！果然是一分錢一分貨。


    • 好玩好玩！左上圖是測我的 ipod，寫入速度怎麼比讀出速度快？右上圖則是我的一顆六年前買的 200G IDE介面硬碟，果然硬碟還是最快的。

    • 至於為何選 SanDisk 這個牌子，因為我手邊只剩下這個牌子的沒被我摔壞過，所以我覺得 SanDisk 比較好。
