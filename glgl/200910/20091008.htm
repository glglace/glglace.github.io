  • 什麼是 Ezgo7？
    • 簡介
      • EzGo7 是『教育部校園自由軟體應用諮詢中心』為校園自由軟體推動所製作的推廣光碟。使用 EzGo7 光碟開機後，您將可以體驗到自由軟體世界的美妙。
      • EzGo7 以 Ubuntu 9.04 為基礎，加上各種教學上好用的自由軟體，並收錄許多自由教材。
      • Ubuntu 是一種Linux 發行版 (distribution)，而 Linux 是一種電腦作業系統，由於 Linux 是自由軟體，人人可以自由的分享與散佈，不同的人，例如個人、一個組織、或一家公司，會把Linux作業系統核心、桌面環境、各種自由的應用軟體、安裝到 電腦的安裝程式等集結起來發行，以方便使用者利用，這就是所謂的Linux發行版。
      • Ubuntu的名稱來自非洲南部祖魯語或豪薩語的「ubuntu」一詞，意思是「人性」、「我的存在是因為大家的存在」。Ubuntu是由南非人，馬克·舍特爾沃斯 (Mark Shuttleworth)創立，第一個版本於2004年10月20日發佈，隨後每6個月發布一個新版，它強調易用性和國際化，以便能為盡可能多的人所 用。

  • 如何使用 Ezgo7光碟開機？
    1. 首先您要有一片 Ezgo7光碟，是免費合法的喔！以下方法三選一：
      • 索取 EzGo7光碟
      • 下載 Ezgo7光碟的 ISO檔，然後自行燒錄
      • 或者是直接跟我索取！我有五百片喔！
    2. 接著，您要更改您的電腦的 BIOS設定， 以光碟機為開機第一順位，如果您完全沒有概念，您可以：
      • 問您身邊的人，「請問您會不會設定用光碟開機？」然後請那個人幫您改，大概只需要花他兩分鐘以內的時間。
      • 如果您想自己動手，可以參考這篇
    3. 將Ezgo7光碟片放入您電腦的光碟機中，開機！


  • 開機速度視您的電腦等級而定，我測試過的電腦，大致上都不會超過三分鐘就可以開機完成。如果超過三分鐘都沒有動靜， 那有可能是光碟片有問題，換一片光碟試試，但也可能是這一臺電腦無法用Ezgo7光碟開機，要解決的話需要專家，建議您換一臺電腦試試。

  • Free win-win 「自由雙贏」是這一次EzGo7的推廣標語，這裡有詳細解釋。

  • 第一次進入 Ezgo7 的桌面，您首先會注意到的，大概是螢 幕解析度是不是您慣用的設定值，這需要去更改一下。例如我這臺液晶螢幕是17吋的，我習慣看 1024 * 768，所以我需要去工作列上的  [系統] ＝> [偏好設定] ＝> [顯示] ＝> 將解析度改成我慣用的數值。

  • 接著您可能會覺得桌面上有一張便條紙很礙眼，上面有很重要的訊息， Ezgo7 的預設使用者的帳號 ezgo7 密碼 moe，有時候您會用到。
  • 看過之後，您應該會去點 x 想把它關掉，可是您會看到一個提示視窗說「刪除後不能復原」，應該會猶豫一下，其實便條只是可以隱 藏的，只要按一下上圖所示的地方就可以了。

  • 另一個您會注意到的，應該是工作列怎麼跑到上面去了？哎呀！因為很自由嘛！您喜歡工作列在哪裡都是可以的，以後再告 訴您怎麼改，暫時先適應一下。

  • 左上方的「應用程式」選單，就相當於 Windows 中按「開始」=> 「程式集」，裡面有很多軟體可以使用，都是免費的喔！

  • 其實不只是「免費」而已，這些軟體都是「自由」的！
    • 「免費軟體」是指使用者不用花錢就可以取 得的軟體，使用者未必有使用、研究、修改和分享的自由。例如該軟體的原始碼不見得是公開的，所以當使用者想要研究該軟體是如何運作，或是想要改善再利用， 或是想要複製送給別人，都需要徵求該軟體所有者的同意。
    • 「自由軟體」因為授權給使用者有使用、研 究、修改和分享的自由，所以通常可以免費取得。自由軟體也可以賣錢，例如市面上有人分門別類整理很多自由軟體在一片光碟上販售，或是一家公司花錢請人將特 定的自由軟體修改成更適合自己的用途，這都是可以的。
