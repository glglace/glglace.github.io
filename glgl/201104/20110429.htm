  • 按右鍵縮小照片，並把照片的「修改日期」 改成 Exif 中「拍照日期」
    1. 開啟終端機視窗
    2. 切換目前所在的目錄
      cd /home/目前使用者名稱/.gnome2/nautilus-scri­ pts/
    3. 開啟我第954回講的 resize_jpg 的命令稿
      sudo gedit resize_jpg
    4. 請加入一行：
      #! /bin/sh
      while [ $# -gt 0 ]; do
      picture=$1
      jpgfile=${picture%%.*}_small.jpg
      /usr/bin/convert -quality 80 -resize 2400x2400 "$picture" "$jpgfile"
      jhead -ft "$jpgfile"
      shift
      done
    5. 將resize_jpg 命令稿存檔
    6. 回到檔案瀏覽器
    7. 選取一堆照片
    8. 按右鍵 => 命令稿 =>點選resize_jpg
    9. 噹噹～ 拍照日期又回來啦！
