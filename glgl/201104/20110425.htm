  • 上回談到在 Ubuntu 擷取螢幕畫面的困擾，在寫上回時，我嘗試使用另一個螢幕截圖工具 shutter，可以滿足我目前想得到的需求。

  • Shutter 在哪裡？
    • 主選單=>附屬應用程式=>擷取圖片_Shutter
　

  • Ubuntu 預設按下鍵盤上的 Print Screen 按鍵時會呼叫 Screenshot 來儲存螢幕截圖，可以改成呼叫 Shutter 來儲存螢幕截圖



  • Shutter 預設會自動儲存螢幕截圖
  • 可以自訂要將截圖存在哪，我習慣先存放在桌面
　

  • 每一個截圖都可以用 Shutter 內建的簡單編輯工具 Shutter Drawing Tool 來加上註解文字(打字或手寫)、醒目圖案、箭頭、圓框方框、號碼圈圈、打馬賽克等等，而且都是物件化的，可以搬來搬去，真是太方便了！
  • Shutter Drawing Tool 也可以做裁剪
　

  • 但我發現在 Shutter Drawing Tool 中無法縮小截圖，沒關係，離開時存檔後回到 Shutter 畫面，在截圖上按右鍵，可以叫出 plugin 外掛出來，其中就有個 resize可以用，真是太貼心了！讚啦！
