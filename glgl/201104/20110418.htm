20110418-- 嘰哩呱啦ACE--第946回--自由的風-2-讓 persistant 生效
  • 昨 天真是個美好的星期天，早上兒子被以前的保姆帶出去玩，女兒則是吃過午飯去洗澡然後在和室靜悄悄的睡著了，哈哈真是寧靜呀！家有小小孩，經常是假日比上班 還累，偶爾清閒一下真是賺到了。到了傍晚兒子陪我出去跑步經過和平快樂田園到和平國小折返，最近手指側面長好多小泡泡，想去流流汗排掉水分，走走跑跑的， 兒子還會找話題跟我聊天，比如說，拔～你會不會十個一數數到一千之類的，真是好玩。
  • 接續第945回--自由的 風-1- Persistent Live USB 製作，3/27做好一隻後，在家裡兩臺電腦、學校三臺電腦上使用都很OK，但一直沒去測試是否真的能夠儲存我對系統所做的更改變動？

  • 4/16早上將我用第945回所述的方法 做好的行動碟，插入我家裡的一公升小 電腦。

  • 按下電腦電腦開關後，要馬上去按F12叫出開機選單，然後選擇用USB開機，我這臺是選「USB:Multiple Card」，但不同電腦會出現不一樣的文字，要自己判斷。
    　
  • 出現 Ubuntu 的開機選單，我選第一個選項，開好機後我做了三個測試：
    1. 新增一個檔案檔儲存在桌面
    2. 設定ADSL連線帳密
    3. 增加chrome圖示到桌面的面板上
      　
  • 接著關機又重開機進來看，發現以上所做的三件事情又全部都不見了！這樣如果我想帶著「20公克的筆電」趴趴走，就不 是很方便了。
    　
  • 第945回所提用 Universal USB Installer 做好的隨身碟會被命名為 PENDRIVE，在 PENDRIVE中找到 syslinux 資料夾，然後用自由軟體 Notepad++來啟其中的 syslinux.cfg 檔案。
  • syslinux.cfg 是用來製作開機選單的，我檢查第一個選項 live 中的指令，看來並沒有 persistant 的命令在其中。
  • 參考過syslinux 資料夾內的 text.cfg 檔案後，我嘗試著在 label live 這一行的上面，增加以下幾行：
    label persistent
    menu label boot the System rewritable
    kernel /casper/vmlinuz
    append file=/cdrom/preseed/ubuntu.seed boot=casper  initrd=/casper/initrd.gz noprompt cdrom-detect/try-usb=true persistent splash --
  • syslinux.cfg 存檔後電腦重開機，果然開機選單中就多了一個我自訂的選項，進入後做同樣的三項測試，然後重開機進來看，結果我所做的變動都還在，哇！成功了！真是開心，想點首那英的「不管有多苦」給澤明聽。
