  • 以下是我第一次使用 Remastersys 來自製 ubuntu 安裝用 iso檔的紀錄：
    • 開始嘗試 remastersys (17:24)
    • 做好 iso 3.5GB用 universal usb installer 來製作 persistant live usb (19:32)
    • 基於阿剛的 20110406 iso 是2.8GB，加上此噗中上述的所有變更 (19:33)
    • 開始做 persistant live usb，老婆叫我下樓吃飯 (19:39)
    • 吃飽了，上去看看進度 (19:56)
    • 做好了，拿persistant live usb來開機的確有成功，但只是區域性成功 (20:14)
    • 結果有些有保留成功，有些沒有，目前發現的是： adsl 連線設定消失了、詞音輸入法 shift 切換消失了 (20:17)
    • 看阿剛那 篇我在想，為什麼不要把 目前使用者家目錄底下所有檔案都cp 道 /etc/skel 這樣不是一了百了嗎？ (20:18)
    • 官網上有關於 dist 選項的說明 (20:20)
    • 有人是全cp過去的  (20:55)
    • 這篇裡有一些使用指南的中英 對照 (20:59)

  • 決定重來一次 remastersy + universal usb installer (21:41)
    • 執行 remastersys clean
      • 可以把 /home/remastersys 裡面上次的東西全部清掉(21:41)
    • 從頭檢檢視看哪些沒有從 /home/xxx 放到 /etc/skel (22:24)
    • 設定 ADSL 連線 (22:24)
    • 詞音輸入法的中英切換鍵改為左邊 shift (22:25)
    • Dock (22:25)
    • 就這3個 (22:29)
    • 稍早從 /home/xxx 放到 /etc/skel 的是 .gconf .gconfd .gnome2 .config .local .SMART Technologies Inc 這幾個 (22:34)
    • 還有我把 光影魔術手和 photocap 的資料夾也放到 /etc/skel/Desktop (22:34)
    • 所以還要加 .gcin 才能儲存剛剛詞音輸入的設定(22:40)
    • .config /google chrome 裏面清掉 (22:40)
    • sudo chown -hR root:root /etc/skel (23:00)
    • .config .gcin 複製過去 /etc/skel (23:07)
    • .config/google chrome .mozilla 兩個整理過後重新傳一次 (23:17)
    • 好了 要來 remastersys 了 (23:17)
    • 打包時去洗碗 (23:35)
    • remastersys做好iso檔了 (00:04)
    • persistant live usb 隨身碟做好了 (00:26)
