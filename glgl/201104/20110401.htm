20110401--嘰哩呱啦ACE--第935回--資訊課-Scratch-15-飛吧！貓頭鷹小子-1
  • 在上下班的車上，兩個小孩都喜歡聽故事CD，我今天拿起一個十片裝的CD盒來看看哪一片還沒有聽過，其中有三、四片，我看故事名稱完全沒印象，可是小孩卻對故事裡面的內容清楚的很，也許是我在開車時把故事當作背景聲音，但他們卻是聽得很仔細。
  • 最近校內有貓頭鷹成功的孵育了四隻小寶寶，全校師生都共同守護著，我在想怎麼讓這可貴的生態觀察與生命教育體驗，也能夠在資訊課中做一點點關聯。
    　
  • 校內老師和校長利用數位相機和數位攝影機逐日補捉小貓頭鷹成長的珍貴畫面，給好奇的學生們能做近距離的觀察，盡量減少打擾他們的次數。然而貓頭鷹晚上在做些什麼就沒辦法追蹤了。
    　
  • 我想到可以上網找看看有沒有相關的影片，我找到吉峰國小領角梟家族的成長影片，覺得幼鳥吃老鼠和幼鳥學飛這兩件事很有意思，可以試試看用 Scratch 來做相關的動畫或遊戲。
    　
  • 於是我的Scratch的官網上用 owl 當關鍵字找找看有沒有相關的作品/專案，然後又發現直接點 owl 標籤來看更有效率，找到了六個我覺得不錯的，但也花了半個小時，雖然很想再看下去，但時間有限就此打住。
    • Scratch 專案 Owl flight cycle
    • Scratch 專案 Owl fight animation
    • Scratch 專案 Owl swoop game
    • Scratch 專案 Mechanical Owl
    • Scratch 專案 Sunset Flight
    • Scratch 專案 The Owl and the Plane

  • 這六個專案中，我最喜歡The Owl and the Plane，因為畫得實在太可愛啦！

  • 這個貓頭鷹「角色」的八個「造型」，如果要學生畫的話，可能要畫個兩三節課也畫不完，不如直接拿來用吧！我把裡面除了貓頭鷹和鳥窩以外的角色都移除，並且把所有的「程式積木」通通移除，我想要讓學生們自己想想，要怎麼表現貓頭鷹的動作。

  • 首先，我示範怎麼讓貓頭鷹眨眼睛，學生跟著做，他們會學到怎麼樣用鍵盤上的特定按鍵來讓 Scratch 中的角色產生我們想要的動作。學生看了之後覺得閉眼睛的時間太久了，我說不一定要「等待1秒」啊！等多久你們可以自己改。

  • 接著，我問學生會不會很想看貓頭鷹飛起來的樣子？我們一起來做做看吧！我只示範了怎麼讓貓頭鷹向上飛，其他向左、向右、向下飛，我要他們自己想辦法利用現成的造型來製作出來。

  • 學生們都好興奮！比我想像中的還要熱烈許多，我起先還擔心說他們會覺得太幼稚，不..不只是熱烈，簡直是著迷！我要他們四個方向鍵都能控制貓頭鷹飛行了之 後給我看，有的學生想的出來，其他人就會去問他是怎麼做的，然後回去做之後發現不行，他們甚至還會互相 debug 呢！學生們想出來的動作，果然跟老師想的不一樣。


  • 過了一會兒，有個學生新增了一個角色「帽子」，並把帽子戴在貓頭鷹的頭上，看起來很可愛！另一個學生新增了一個角色「沙灘球」，並放在貓頭鷹的頭上，看起好像會耍特技！我靈機一動，問學生想不想要讓貓頭鷹飛來飛去，碰到帽子的時候，就把帽子戴上飛走？學生反應非常熱烈，然後大家都想學，接著就很熱烈加上各種道具，大家玩得好高興！

  • 這是足球的程式，這樣就能讓足球被貓頭鷹碰到的時候，跟著貓頭鷹走了。

  • 不過要特別注意的是，為了要讓「足球看起來是在貓頭鷹頭頂上」的效果，要去改變足球造型的中心位置，從球心移到求的下方。
  • 有個學生還讓貓頭鷹戴上假髮、太陽眼鏡，大家都來圍觀！看到學生認真的投入、討論、開心地大笑，感覺真是棒透了！他們的想法和做法彼此互享學習、吸收與變化，真的是有快樂學習的感覺。

  • 這一節課成功的經驗，讓我注意要時常提醒自己，Scratch 教學活動，老師不要把上課教學時間用到滿，要留下充足的時間師生「即興的」互動與創作。
