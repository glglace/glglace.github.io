  • 本回紀錄的是，4/11我將阿剛老師打包的TPET in ezgo8 mac主題重包版 ，安裝到電腦教室的學生電腦硬碟中，做成 Ubuntu 和 Win7 多重開機。
    　
重新劃分電腦教室學生電腦硬碟 的磁碟區
  • 為什麼要安裝到硬碟？
    • 安裝到硬碟有其困難度，一不小心可能會毀掉電腦裡面的所有資料，但比起用光碟或隨身碟來使用 Ubuntu，效能好很多！
      　
  • 這臺電腦送來學校時就已經預先做好 C: 和 D: 兩個分個區，我要將 D: 的部分切割出一部份來給 Ubuntu 使用。步驟如下：
    • 開啟電腦，以Teacher身份登入 Win7
    • 複製D:中要保留的檔案到C:上
    • 開啟管理介面
    • 刪除D:
    • 先增加一個磁碟區 30597MB，不格式化，準備給 Ubuntu用
      • 注意！要用來安裝 Ubuntu的分割區最好不要是數個分個區中的最後一個，我一年多前有次將 Ubuntu 安裝在硬碟的最後一個磁碟區，但過幾天後該磁碟區莫名奇妙消失，查了一些資料說大容量硬碟最好把 Ubuntu 作業系統安裝在比較前面的磁碟區。
    • 再增加一個磁碟區200000MB，格式化NTFS
    • 刪除磁碟區 30597MB，這樣等一下安裝 Ubuntu 時比較方便。
    • 把剛剛複製到C:的資料再搬回D:
    • 關機

  • 以下是上述步驟的圖解，4/12 請三個學生義工來幫忙這個部分，大約20分鐘就做好20臺，我很開心，學生也很有成就感。

































　
將TPET in Ezgo8安裝到硬碟
  • 插入我第945回 製作的 Live USB 隨身碟，您也可以下載 ezgo8 的 iso檔來使用


  • 電腦一開機就馬上選 F12，然後選 USB 隨身碟開機
  • 您電腦和隨身碟不見得與我的相同，所以可能會看到不一樣的文字。


  • 選 install
  • 選中文繁體


  • 確定無誤就按「向前」，意思就是「下一步」啦！
  • 這裡很重要！要非常小心！
  • 選擇「使用最大的連續可用空間」，就是剛剛我刪除的那個30597MB磁碟區。


  • 輸入自訂的使用者帳密
  • 選擇要不要匯入同一臺電腦中 windows 使用者的資料


  • 前置步驟全部都做好了，勇敢的選擇「安裝」吧！
  • 安裝過程大約20分鐘。
