20110408--嘰哩呱啦ACE--第940回--自由的風-3-用Wine跑光影魔術手
  • 我一直很想「完全的」換到 Linux 作業系統上做我日常會用電腦做的事，可是總有一些我常用的軟體功能是 Linux 上的軟體所沒有的，例如光影魔術手就是其中之一，今天很高興能夠順利的在Ubuntu中用 wine 來執行光影魔術手，澤銘，我又接近您一點點了。
  • 電腦教室的這一批學生電腦，是去年10月更新的，詳情請見我的網誌第835回和第836回。電腦送來時就已經預裝了 Windows 7，為了要給學生體驗自由軟體的好，當時我說等驗收後要來安裝 ezgo8，沒想到一拖就是半年，真是慚愧。
    　
  • 今天早上到學校電腦教室找一臺學生電腦來安裝 Ubuntu Linux 作業系統，我使用的是阿剛老師打造的 TPET in Ezgo8 mac介面版，20110406 釋出。
    　


  • 用學生電腦pc20來測試：
    • 8:30 先進 Windows 7 去看一下目前的磁碟分割，我決定把D：裡的檔案先剪下貼上到C：，然後把D：分割區225GB 刪除，再重建D：給它150 GB就好，把檔案搬回來。
    • 08:35 電腦重開機馬上按 F12，選擇光碟開機、選擇直接裝到硬碟去，選擇「使用最大的連續可用空間」，有75GB，這樣程式比較簡便。
    • 08:37 做 ezgo8安裝前設定
    • 08:45 開始安裝程式
    • 09:10 裝完重開機
    • 09:15 設定網路連線
    • 09:20 去contro center 跑更新管理員，但太新了，沒有找到需要更新的
    • 09:30 執行 lazyezgo 除預選項目外，我選擇加裝 teamview, dosbox、模擬交通、壓縮格式支援，但發現有一些阿剛已經做過了。
    • 09:50 試玩一下「模擬交通 simtrain」遊戲，但不小心被我弄成別的語言介面，我不知道改回來，放棄。
    • 10:00 試試看 team viewer 怎麼介面有點簡繁混雜，另用一臺 Win XP 嘗試與之連線，但很搞笑的是 win 版的也被我裝成不知道什麼語言的介面，還好有中文版可以對照位置找出語言選項在哪裡。
    • 10:20 再玩下去就沒完沒了，趕快來測一下光影魔術手，這樣就要花掉一個早上了，適可而止吧！
      　

  • 我將 nEO_iMAGING_31158_Portable 放在隨身碟，插入到電腦中，桌面上會跳出隨身碟的圖示，點兩下開啟「檔案瀏覽器」，找到執行檔 nEOiMAGING.exe我就習慣性地點兩下想執行，忘了是要按右鍵選擇「以wine開啟」，不過很驚喜的是，點兩下似乎也行！實在是太方便了。
    　

  • 由於是新電腦，跑什麼都很順、很快，像瀏覽器、文書處理、試算表、簡報這些的，按一下圖示之後，三秒鐘左右軟體就打開了，真的是很讚！
    　
  • BUT！人生就是這個 BUT！我實際做一張照片的縮圖，發現同一張照片經過同樣的處理程式，在 winXP和 用 wine 做出來的照片，效果有差！用 wine+光影魔術手，加邊框和縮圖後，看起來覺得有點糊糊的。不知道有甚麼辦法可以改？
    　


  • 10:31 我發噗問人，阿剛老師說：把wine升到1.3試試，sudo apt-get install wine1.3，我試了，結果沒什麼差別，還是糊糊的。
    　
  • 很傷心地去吃中飯，午休時間還是一直再想這個問題該怎麼解決？13:40 靈感來了，我想到在光影魔術手中做縮圖的時候，預設是「自動選擇合適的採樣方法」，如果手動指定的話，效果會不會好一點？嘗試 Lanczos3，因為有附註「縮小時推薦」，果然沒錯！不會糊糊的了！耶！萬歲！
    　

  • 阿剛老師打 造的 TPET in Ezgo8 mac介面版很漂亮，嘗試著把光影魔術手拉到下方的 Dock 去，可是都失敗，嘗試新增一個左方Dock 可以順利拖進去，這樣用起來比較方便，不用在檔案瀏覽器裡面一層一層的找。不過寫到這裡時，瞄了一眼上圖，疑？有呀，下方Dock從左邊算來第二個不就是 了嗎？唉~ 一定是眼睛太累了，該休息休息。


  • 我今天是斷斷續續的做，眼睛並沒有一直盯著電腦螢幕，為什麼會覺得眼睛很疲累呢？我發現這臺學生電腦的位置，剛好前面是窗，後面也是窗，應該是從外面進來的前後光線讓眼睛很不舒服，趕快去把燈打開，感覺好像有好一點，不過還是到此為止吧！

　 　 　 　 　
  • 280分鐘完成本回。
