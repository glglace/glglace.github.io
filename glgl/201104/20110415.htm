  • 3/26看到教育噗浪客的阿剛老師發了一個噗：檔案分享TPET in ezgo8 mac主題重包版 可自訂使用者名稱。在該噗中小P老師問要如何安裝在隨身碟上？阿剛老師提供了一個方法：T客邦：靠一支隨身碟，Ubuntu系統、檔案、設定全包，我很有興趣想嘗試看看。
    　
  • 我有一個閒置的8G隨身碟，正好拿來試試：
    1. 下載阿剛老師打包的TPET in ezgo8 mac主題重包版
      檔名是 TPET_in_ezgo8_menu_mac_1000325.iso
    2. 到 Universal USB Installer – Easy as 1 2 3 網頁中去下載「Universal USB Installer」，這個軟體是採用 GNU GPL 授權的自由軟體。
    3. 執行「Universal USB Installer」
    4. 以把 TPET in ezgo8 mac主題重包版裝到8G隨身碟為例：
      • 步驟一 Step1 選擇要哪一種 Linux 發行版
        • 下拉選單中選擇 Ubuntu 10.04.X
      • 步驟二 Step2 指定 iso 檔所在路徑
        • 這裡對 iso 檔的檔名有嚴格規定，所以請將TPET_in_ezgo8_menu_mac_1000325.iso
          改名為 ubuntu-10.04-desktop.iso
      • 步驟三 Step3 選擇USB隨身碟所在的磁碟機代號
        • 這裡要很小心，千萬不要弄錯！
        • 選擇要 format 格式化，注意！隨身碟上現有的資料都會不見喔！
      • 步驟四 Step4 選擇 Persistant file 的大小
        • Persistant file 的用途是用來儲存您所做的設定和檔案，如此一來就不會每一次開機都還原回預設狀態，這是與 Live CD 最大的不同。
        • 最大可以設到 4G，以8G隨身碟為例，建議設3G，因為阿剛老師打包的這個 iso檔裝到隨身碟上大約會用掉4.3G。
      • 按下 Create 按鈕，開始製作。
