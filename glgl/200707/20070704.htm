  1. 聲音

    • Windows 錄音機

      • 神奇的演奏

      • 如何利用麥克風和電腦錄音

        1. 在電腦主機背後的音效卡三個孔中找到麥克風插孔，將麥克風插頭插入。

        2. 在電腦上音量控制（Volume Control）視窗中選擇錄音來源

        3. 開啟錄音軟體：點選螢幕左下角「開始」鍵→「程式集」→「附屬應用程式」→「娛樂」→「錄音機」，開啟Windows內建錄音程式。

        4. 錄音時的注意

          • 觀察聲波的曲線調整自己的音量, 如果看不到聲波曲線表示1,2兩步驟出了問題

          • 盡量減少外來噪音

          • 調整麥克風的位置並測試是否會錄到呼吸聲及爆裂音

          • 按下錄音鍵後不要急著開始錄音, 大約等三秒再開始

          • 錄音時間一口氣最長不能超過一分鐘，但可接續著錄

    • 編修照片

      • 改變大小

        • 使用數位相機拍照，設定越高的畫素當然就能拍得越好，但是若有需要轉寄給別人、製作網頁及插入的檔案中的時候，也許就會希望檔案小一點比較好。
          　
        • 介紹一個縮小照片的免費工具， Microsoft Windows XP 的 PowerToys 裡面的 image resizer，安裝好了之後，在檔案總管中只要選取一張或多張照片，然後按右鍵，選擇 Resize Pictures，會有四種預設的大小可以選擇，如果都沒有中意的，可以按 Advanced 按鈕自訂影像大小，如此就可以輕鬆的轉換影像大小。
          　
        • 上述軟體很好用，可惜不知為何，有的電腦安裝之後，在檔案總管中選取一張或多張照片，然後按右鍵，並不會出現 Resize Pictures，所以另外推薦一個也是免費的軟體Virtualzone - JPEG Resizer
          • 免費的軟體不代表沒有著作權，只是指可以免費使用而已

          • 做A4檔案，一頁六張照片：縮小到 640*480畫素
