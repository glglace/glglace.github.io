20100202-- 嘰哩呱啦ACE--第697回--電子白板教學應用-60-需讀外部資料的flash
  • 從1/15因處理學期末事務、寒假研習、教育噗浪客第一屆年會，我的部落格暫時停筆。昨天 2/1 才開始來把700回的最後五回寫一寫，應該可以在過年前完成。
    • 上回提到我和教育噗浪客雄哥合作的「注音認字日記」螢幕鍵盤開發經驗，如果您想玩玩看，可到雄哥的部落格Flash : Phone 注音小鍵盤一文中，找到下載的連結。
      • phone.swf ：主程式
      • phonetab.txt ：注音、國字、標點符號的對照表

    • 下載到您的電腦後，請記得：
      • 這兩個檔案一定要放在同一個資料夾
      • 資料夾的名稱和路徑中不能有中文字

    • 如果您想用 SMART Notebook 或 VOSA Trek，整合到您的電子白板數位教學檔案中，會有一點點困難，但很好解決，您可以這麼做：

    • SMART Notebook (我用的是 9.7版)
      1. [插入] => [多媒體動畫檔案] => phone.swf => [開啟]
      2. 然後您會發現無法使用phone.swf
      3. [檔案] => [儲存]=> 暫時存在桌面，檔名命名為：注音認字日記.notebook
      4. 關閉「注音認字日記.notebook」，回到您的電腦桌面
      5. 將「注音認字日記.notebook」重新命名為「注音認字日記.zip」
      6. 打開「注音認字日記.zip」，進入 flash 資料夾，尋找phone.swf
      7. 將 phonetab.txt 放到與 phone.swf 相同的資料夾中
      8. 關閉「注音認字日記.zip」，重新命名為「注音認字日記.notebook」
      9. 開啟「注音認字日記.notebook」，檢檢視看有沒有成功！

    • VOSA Trek (我用的是 V5.1.5C 教育版)
      1. [插入] => [多媒體檔案] => phone.swf => [開啟]
      2. 然後您會發現無法使用phone.swf
      3. [檔案] => [儲存檔案]=> 存在桌面，檔名命名為：注音認字日記.trekx
      4. 關閉「注音認字日記.trekx」，回到您的電腦桌面
      5. 將「注音認字日記.trekx」重新命名為「注音認字日記.zip」
      6. 打開「注音認字日記.zip」，進入 attachments 資料夾，然後進入 Flashs 資料夾
      7. 將 phonetab.txt 放到與 phone.swf 相同的 Flashs資料夾中
      8. 關閉「注音認字日記.zip」，重新命名為「注音認字日記.trekx」
      9. 開啟「注音認字日記.trekx」，檢檢視看有沒有成功！

    • 依此類推，若您有其他需要讀取外部文字檔的 flash 想要整合到您的電子白板數位教學檔案中，不論是用SMART Notebook 或 VOSA TrekV5，都可用以上方法來解決！例如雄之語文高手裡面的許多教學flash，都是可以自訂題庫的喔！

    • 很妙的是，如此一來，就沒有flash檔案路徑中有中文字時無法執行的問題了！
