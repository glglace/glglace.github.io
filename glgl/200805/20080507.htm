  • 今天接續第356回講的，如果短時間不用電腦，應該要設定電腦於「待命」狀態可以省電，5/5週一教師晨會時，總務主任報告說下班時最後一個走的人已經很辛苦了，還要檢查保全設定和檢查辦公室公用電腦是否有關機實在是很可憐，希望大家能夠下班要走的時候思考一下等一下是否沒有人會再使用電腦了，盡量能關機再走。
    　
  • 可是有時候真的很難判斷到底要不要關機，所以我想找一個辦法，讓電腦能夠在，比如說下午 5:30的時候自動關機。我第一個想到的是windows xp裡面有一個「排定的工作」，可以設定電腦到某個時間自動做某個動作，不過這個方法有個問題：如果在下午 5:30的時候還有老師正在使用電腦，老師想要中止關機的話，必需在 [開始]＝> [執行]下指令 shutdown -a，可是要所有老師都記住這個指令是不夠人性化的，而且可能會手忙腳亂。
    　
  • 在網路上尋覓了三、四種方式，找了一個小時之後，我覺得3Cbank部落格--如何讓電腦定時「自動關機」此文中介紹的軟體不錯，實際測試也很合用，在此推薦給大家使用。
    　
  • 自動關機程式：Shutter
    • 官方網站-- [den4b] - Denis Kozlov
    • 授權方式-- 創用CC 3.0 姓名標示-非商業用途-禁止改作
    • 軟體介紹-- Shutter-PChome Online 網路家庭下載Toget
    • 使用說明-- 3Cbank部落格--如何讓電腦定時「自動關機」
    • 我的經驗--
      • 軟體是英文介面，網路上有人嘗試中文化，可是我覺得作者已經聲明「禁止改作」，所以要中文化至少要取得原作者的同意。
      • 下載安裝好之後，我是這樣設定的
        • Event 選擇 On Time
        • Time 設定 17:30:00
        • Action 選擇 Shutdown
        • Options
          • General 頁面中要打勾的選項為
            • Autorun at Windows startup/logon
            • Start event when program starts
            • Minimize when click on close button
          • On Action 頁面中
            • Show Message要打勾
            • 時間設定 600秒，免得老師手忙腳亂或只是去上個廁所，當然也要事先跟老師們告知。
            • 輸入提示文字，「電腦即將關機，要取消請按 Cancel 按鈕」以免老師們不知道如何取消自動關機。
