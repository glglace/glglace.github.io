  1. 簡介
    • 由芬蘭的 Kuopio大學化學系所開發
    • 是採用GNU GPL 授權的自由軟體
    • 官方網站 http://www.bioinformatics.org/ghemical
      　
  2. 軟體說明 (資料來源: Wekeywiki)
    • Ghemical是3D化學繪圖軟體，這個程式可以在輸入原子種類、數量及鍵結條件之後，畫出分子的的立體結構圖。適閤中學老師講解化學分子的立體結構，學生也可以自行練習。Ghemical也適合用在大學有機化學的課程，特別是講解穩定的分子構形與測量原子間的距離；對於大學生化課程， Ghemical可以輸入胺基酸或核酸，馬上可以看出立體結構。在Ghemical畫完圖後還可以存檔，內建格式是gpr。
    • 同學們在上化學課時，可以用Ghemical等程式來幫助自己理解分子的立體結構，化學老師們也可以用Ghemical來進行教學。只要學會了基本操作，就可以享受Ghemical的便利！

      • 在芬蘭的 Kuopio大學的下載網址
    • Windows

  4. 教學活動體驗
    • 畫出三聚氰胺 (C3H6N6) 的分子結構
      1. 在作圖區中任一處點滑鼠右鍵，在跳出式選單中，選擇  Render => Label Mode (標籤模式) => Element (元素)，這樣等一下畫原子的時候會標 示出元素名稱。
      2. 先在心裡面想著要在作圖區畫一個六角形，假設六個頂點為1,2,3,4,5,6。
      3. 點一下工具列最左邊的鉛筆 d (draw)， Ghemical預設是畫碳原子。
      4. 在作圖區中你想像的六角形頂點 1,3,5的位置點三下，這將會畫出三顆碳原子，稱之為 C1、C3、C5。
      5. 點一下工具列中的週期表 el，在週期表中選N(氮)。
      6. 在作圖區中你想像的六角形頂點 2,4,6的位置點三下，這將會畫出三氮原子，稱之為 N2、N4、N6。
      7. 從六角形中的三個碳原子向外分別再畫出三個氮原子。
      8. 點一下工具列中的原子鍵類型 bt (bond type)，再點一下Single(單鍵)，在C1上按滑鼠左鍵不放，拉到N2後放開，這將會畫出一個單鍵，請依序用單鍵連結 C3和N4、C5和N6。
      9. 點一下工具列中的原子鍵類型 bt (bond type)，再點一下Double(雙鍵)，在C1上按滑鼠左鍵不放，拉到N6後放開，這將會畫出一個雙鍵，請依序用雙鍵連結 C3和N2、C5和N4。
      10. 點一下工具列中的原子鍵類型 bt (bond type)，再點一下Single(單鍵)，從三個碳原子分別連向六角形外的三個氮原子。
      11. 在作圖區中任一處點滑鼠右鍵，在跳出式選單中，選擇 Build => Hydrogens => Add，Ghemical將會自動補上氫原子。
      12. 在作圖區中任一處點滑鼠右鍵，在跳出式選單中，選擇 Compute => Geometry Optimization (幾何位置最佳化)，Ghemical將會自動計算出原子之間的距離和角度。
      13. 點一下工具列中的運行軌道 o，然後對著三聚氰胺分子按滑鼠左鍵不放，稍微拖曳一段距離後，放開滑鼠左鍵，這樣就可以感受到化學分子的立體感！
