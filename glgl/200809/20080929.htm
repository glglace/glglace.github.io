  • Scratch 軟體何處下載？
    • 官方網站 http://scratch.mit.edu/
    • 目前最新版本 Scratch 1.3 (2008年9月2日釋出)
    • 可免費下載，下載網址為 http://scratch.wik.is/Scratch_1.3_Download
      1. Windows 2000, XP, Vista 使用的安裝檔 ScratchInstaller1.3.exe
      2. Windows 2000, XP, Vista 使用的免安裝檔 WinScratch1.3.zip
      3. 蘋果電腦 Mac OSX 10.4 或以後版本 MacScratch1.3.dmg
      4. Linux
        • 據說可用  WINE 來執行之，但我還沒嘗試過
        • 非官方版本 Scratch 1.2 for Debian/Ubuntu，請到此網頁下載並有安裝說明。
      5. EzGo6 (教育部自由軟體應用推廣光碟)今年的九月版有收錄 Scratch
        • 下載 iso 檔 （可燒錄到光碟）
          　
  • Scratch 軟體版權為何？
    1. 此程式由麻省理工學院媒體實驗室(MIT Media Lab)的終身幼稚園團隊(Lifelong Kindergarten Group)開發，以及美國國家科學基金會(National Science Foundation)、微軟 Microsoft、Intel基金會、Nokia、麻省理工學院媒體實驗室研究聯盟的支助。
    2. 可免費使用，但是否為自由軟體？Scratch軟體安裝好之後，「說明」視窗中顯示的是「著作權©2008麻省理工學院‧保留一切權利」，但官方網站有提供原始碼下載，有的媒體報導說這是開放原始碼程式 ，實在是令我搞不懂？
    3. 後來在原始碼下載壓縮檔中，我找到了 license.txt，看內容還蠻接近 GNU GPL的授權方式，除了有一點保留...
      • The purpose of these restrictions is to avoid confusion in the Scratch user community between the official version of Scratch (supported by the Lifelong Kindergarten group) and experimental variations. We are happy to share our source code, but we want to protect the Scratch brand and our user community.
    4. 結論！Scratch 軟體可以自由的使用、研究、修改和分享，但非官方修改過後的版本，其命名中不得含有「Scratch」字眼，以免與 MIT的官方版本混淆。
