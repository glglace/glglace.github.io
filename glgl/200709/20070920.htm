  • 如何處理「編輯衝突」？

    • 聽起來有點可怕，是要吵架還是打架嗎？呵呵，讓我舉個例子吧！

    • 假設有兩個使用者，user A 和 user B。

      • 9點10分

        • user A 在時開始編輯「甲條目」，  user A看到「甲條目」的內容是「1234」，

      •  9點11分

        • user B 在時開始編輯同一個「甲條目」，user B看到「甲條目」的內容也還是「1234」。

      •  9點20分

        • user B 在1234的後面加上了「甲乙丙丁」，按下了「儲存頁面」按鈕，然後就走人了。

        • 此時「甲條目」的內容是「1234甲乙丙丁」。

      • 9點30分

        • user A 在1234的後面加上了「忠孝仁愛」，按下「儲存頁面」按鈕，這時候，user A 就會遇到一個「編輯衝突」！

        • user A 看到「編輯衝突」的提醒

          • 有人在你開始編輯後更改了頁面。上面的文字框內顯示的是目前本頁的內容。你所做的修改顯示在下面的文字框中。你應當將你所做的修改加入現有的內容中。 只有在上面文字框中的內容會在你點擊"儲存頁面"後被儲存。

      • 9點31分

        • user A看到上面的文字框內的內容是「1234甲乙丙丁」，於是他知道有另外一個人也正在編輯「甲條目」 。 他把自己新增的東西從下面的文字框搬移到上面文字框，然後再按一次「儲存頁面」按鈕。

        • 此時「甲條目」的內容是「1234甲乙丙丁忠孝仁愛」。
