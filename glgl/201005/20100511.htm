    • 教育部重編國語辭典修訂本
      • 電腦是一種能接受資料，並按照人為指示的方法將它處理，並產出有用結果的裝置。通常由輸入設備、輸出設備、中央處理器、主記憶體及算術與邏輯單元等五大部門所組成。
    • 中文維基百科的電腦條目
      • 電子計算機（英語：Computer）是一種利用電子學原理根據一系列指令來對數據進行處理的機器。
    • 英文維基百科的 computer 條目
      • A computer is a programmable machine that receives input, stores and manipulates data, and provides output in a useful format.

  • 如果擺三種不同機器的照片要學生挑出「電腦」，我想應該90%以上的學生都沒問題。但是要用30個字以內解釋「電腦」這個語詞給國小二年級的學生聽得懂，還真是有挑戰性。目前我還沒想出要怎麼撰寫這30個字。

  • 對小學二年級的學生，電腦的組成，我想應該是用可觀察得到的主機、螢幕、鍵盤、滑鼠、喇叭這「五大部門」來說明會比較好吧？談到電腦主機的構造，我認為應該是國小五年級左右比較適合。
