20110615--嘰哩呱啦ACE--第988回--自由的風-36-清理儲存空間
  • 今天又送走了一批畢業生，這次的畢業典禮改由五年級學生來當主持人，臺風相當不錯，還有別開生面的校隊交接儀式，笑中帶淚，典禮流程既有傳統也有創新，感覺十分溫馨。
  • 承接前天的第986回–自由的風-35-磁碟用量分析器，6/12 想清理 ubuntu 裡的垃圾，第一個想到的是「電腦垃圾清道夫」，一執行就自己開始分析系統
    • 系統=>管理=>電腦垃圾清道夫

  • 結果它提供的刪除建議，似乎是根據該軟體有沒有在官方的套件庫，使用者要自己決定該軟體的去留，這對我來說並不實用。例如 scratch 我不想刪除但卻在名單之列。
    　
  • 於是上網找資料，找到一篇自由軟體鑄造場－源碼秘技－清理 Debian/Ubuntu 儲存空間覺得不錯，想照著作作看並紀錄一下效果。
    • sudo apt-get clean
      • 使用量從12GB降到11.8G

  • localepurge
    • 安裝
      • sudo apt-get install localepurge
    • 設定語系
      • 我選擇保留 en,zh,zh-tw 這3個
        　

  • 回報說清出了約 147MB
    　

  • BleachBit (類似windows下的CCleaner)
    • 安裝
      • sudo apt-get install bleachbit
    • 執行
      • 主選單=>系統工具=> BleachBit (as root)
      • 是有清除一些但都不是佔用很大空間的
        　

  • 我又清除 chrome 和 firefox 的快取
    　

  • 另外我又清除了桌面上暫放的資料，這樣總共使用了10.9GB，來remastersys 試試看吧！很遺憾的是，remastersys的backup選項是要在 4.7G以內的 iso檔才能做得出來，看來只好使用 clonezilla來備份了。

　 　 　 　
  • 70分鐘完成本回。
