輔導團影音
  • 這幾天在忙著趕工，想要盡快完成與國小英語輔導員-康壽國小許瑞蘭老師一起合作的一個影音計畫，起先是想用對話的方式，我扮演一個菜鳥老師，她則是經驗老到的輔導員，藉由觀看她的教學錄影，一起討論教學理念，以下是我們一起工作的經過與所需時間的紀錄：
    　
    • 2006-12-15 三個小時
      • 我到康壽國小，先商討如何對話，課本內容六頁掃描起來備用，接著因為場地不合適，干擾聲音太多，換了一個較安靜地點。然後試著用Wink 螢幕錄製軟體錄一段看看，結果發現 Windows media player內的影像出不來，而且聲音斷斷續續，但製作模式大致商討完畢，約定週日到我家做。但因氣候不佳取消。
        　
    • 2006-12-17 一個小時
      • 我自己在家裡研究如何讓Windows media player內的影像出來，查了一些資料，原來是要改一些設定：
        • Windows media player 功能表 => 工具 => 選項 => 效能 => 進階 => 取消兩個「使用重疊」的設定。
      • 接著用Wink順利錄下了Windows media player中播放的影片畫面，可是由於一秒五格的方式去錄，做成Flash檔再播出時，人的動作會斷斷續續，若改為一秒錄二十格，出來的Flash檔會很肥大，而且聲音還是會斷斷續續，大概是電腦CPU和記憶體等級不夠吧！
      • 放棄用Wink的方式，改成用製作影片的軟體「威力導演」來做，終於錄好的片段在觀看時比較順暢了。email給瑞蘭老師看demo，並約好下週二再戰！
        　
    • 2006-12-19 三小時
      • 請瑞蘭老師到永興國小來做，商量過後決定這次不要用老鳥菜鳥的對話方式了，因為準備時間會更長，改由她一人唱獨角戲，我只負責操作電腦。很順利的錄好了21段影片的解說。
        　
    • 2006-12-20 一小時
      • 我試著用「威力導演」做一段，將教學現場錄影放在影片軌，將昨天瑞蘭老師的旁白放在配音軌，然後調整適當的聲音比例，並加上標題背景幕、片頭音樂，做成了一個新的demo片段。
        　
    • 2006-12-21 三小時
      • 早上在輔導團員的聚會中放給其他輔導員看，大家覺得效果不錯。下午一點回到學校後就開始製作。
      • 這次把標題背景幕改用動態的，並分成兩幕，一幕是計畫名稱，一幕是這一小段影片的名稱，換上比較輕鬆的音樂，加上過場效果，越來越像樣了！
      • 做了五小段，是整個工作四分之一的完成度。決定先把目錄網頁做起來(目前還醜醜的)，然後加上影片連結以及每一小段的時間長度，先請瑞蘭老師看一看再做。
        　
    • 想看嗎？在這裡，不過請等到12/21晚上8點以後再看，因為現在已經到了要去保母家接小孩的時間了，今天就到此為止。
