20091214--嘰哩呱啦ACE--第676回--資訊課-做聖誕樹卡
  • 最近有點空檔的時候，就會拿起龍應臺的「大江大海一九四九」來翻個幾頁，看著看著，會很想在過年假期時，多瞭解自己父母親的往事，那一年，我爸爸十五歲，我媽媽十二歲...
  • 上上週三年級資訊課的小畫家單元告一個段落，由於聖誕節快到了，我構想了一個教學活動：讓學生用小畫家作「聖誕樹 卡」！

  • 我先讓學生玩一個韓國 sayhi 網站上的「Making Christmas Tree」遊戲。學生可以從八種聖誕樹造型中挑一種，然後在選一些小裝飾物放上去，其中最好玩的是天使如果放在天空中會飛上去，聖誕老人如果放在天空中則 會用包袱巾當降落傘跳下來。


  • 學生佈置好自己喜愛的聖誕樹之後，接下來就是一些電腦使用的技能教學了。
    • 按下鍵盤上的 Print Screen 按鍵 (我發現伸縮指揮棒在資訊課很好用！)
    • 開啟「小畫家」 => 影像 => 屬性 => 寬度 800 高度 600
    • 在空白的地方，按右鍵選「貼上」
    • 使用「選擇」工具，在聖誕樹外面畫一個框框
    • 在框框裡面按右鍵選「剪下」
    • 小畫家 => 檔案 => 開新檔案 => 不要儲存選「否」
    • 在空白的地方，按右鍵選「貼上」
    • 把聖誕樹拿到畫紙的左邊或右邊
    • 在空白的地方畫圖或寫字給你要送的人

  • 由於三年級我還沒有教中英文打字輸入，我原先構想是讓他們用粉刷工具寫字，不過有些已經會打字的同學開始打字之後， 學生之間會互相教，雖然不在我的構想之中，不過倒是可以作為下一系列英打課程學習的起頭動機。

  • 下次資訊課讓他們打上 Merry Christmas，並在英文字旁用小畫家畫上一些裝飾線條，然後印在漂亮的美術紙上送給他們，讓他們送給自己的父母或爺爺奶奶。

  • 學生一律關掉音箱，由老師電腦放出該遊戲的平安夜背景音樂，教室裡充滿了寧靜柔和的氣氛，真好！
