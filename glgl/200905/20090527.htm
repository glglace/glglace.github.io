20090527--嘰哩呱啦ACE--第578回--軟體使用-您的 Windows XP 是盜版的？
  • 上週偶然的機會，聽一個老師在說他的公務帳號密碼，我覺得密碼怎麼會那麼簡單？原來那是因遺失而補發的密碼，我建議還是去重新設定密碼一下吧！
  • 5/18 上午一個朋友打電話問我說，她的電腦出現盜版軟體的警告，她不敢繼續用，該怎麼辦才好？我問她說當初買電腦的時候有沒有買 Windows XP，她說有，於是我說那可能是電腦公司幫你安裝軟體的時候，使用的是一片快速安裝的光碟，沒有用到你所買的Windows XP序號，才會出現盜版軟體的警告，你先把你的序號找出來，我告訴你怎麼改。
    　
  • 其實講電話的時候我也不會改，我只是知道可以改而已。5/19 早上上網查了一下，有的方法似乎不大合法，有的方法則是很麻煩，有的雖然有提供簡便的程式，但也有網友說擔心是否內含病毒，總之似乎不是那麼容易。
    　
  • 最後在發卡布洛裡面看到一篇更改XP序號的方法 - VBScript 腳本語言，可以看到他所撰寫的 VBScript 的內容，感覺很安全，於是照著做一次就成功了！程式是這樣的：
  • 看到盜版軟體的警告，雖然繼續用下去也不會怎樣，但總是看了很礙眼，您說是吧？請您跟我這樣做！
　 　
  • 在桌面空白處按滑鼠右鍵，選擇新增一個文字檔案。
　 　
  • 打開新增文字檔案，將發卡布洛裡更改XP序號的方法 - VBScript 腳本語言一文中的VBScript複製貼上，並將「您的序號」這四個字，換成您的 Windows XP的光碟的外包裝或盒子上的序號，然後存檔。（有的是直接貼在電腦主機上）
    　
  • 序號一共由25個英文和數字所組成，其中如果有 - 號請去除。
    　
  • 將檔案名稱「新增文字檔案.txt」改成「xp.vbs」，如果您只看得到檔名但看不到附檔名，請您身邊會調整設定的人幫您。
　 　
  • 按兩下 「xp.vbs」這個檔案的圖示，就會執行更改XP序號的動作，執行完畢後，大約10秒左右，您就可以看到原本桌面背景從「昏天暗地」的黑色，又變回原來的「藍天綠地」了，恭喜您！序號更改成功！
　 　
  • 真的就這麼簡單嗎？嗯，的確是蠻令人懷疑的，這樣好了，要是您不相信您已經通過微軟的正版Microsoft軟體驗證的話，您可以點選前述超連結去驗證看看，如果有出現左圖畫面的話，表示您可以放心了。
    　
  • 如果您沒有正版的Windows，也不想花錢去買，建議您考慮改用自由免費的 ubuntu
