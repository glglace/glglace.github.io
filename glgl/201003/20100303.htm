20100303--嘰哩呱啦ACE--第708回--如何打出方塊打勾字
  • 這幾天上下班經過日月潭水社壩堤時，都看到有新人在拍婚紗照，前天有三對，昨天有兩對，哎呀！春天真的來了！
    • 前幾天同事想要把一份調查檔案的電子檔中，有個選項需要填寫一個方塊然後裡面打勾，可是在插入符號時找不到這種字。我花了點時間研究一下，發現原來在 「Wingdings 2」字型中，還真的有這種字，可以用鍵盤的 「shift + r」組合鍵按出來。

    • 不過，各位可以從上圖中看出，方塊打勾這個符號可能會比檔案中的方塊符號小一點(在同一種文字大小設定時)，如果覺得不夠美觀，就要請您再調一下字體大小了。
    • 還有，要記得字型名稱是「Wingdings 2」還好，如果要背「shift + r」才能用，那就痛苦了，可以這樣：
      • 在 Word 中，[插入] => [符號] => [字型] 選 Wingdings 2 => 找到方塊打勾字 => [確定]
      • 在 Writer 中，[插入] => [特殊字元] => [字型] 選 Wingdings 2 => 找到方塊打勾字 => [確定]

    • 如果您想多瞭解 Wingdings 字型，請看維基百科 Wingdings 條目。

    • 如果您想看看用鍵盤可以按出哪些 Wingdings 字和 Webdings 字，請看
      • Wingdings　http://en.wikipedia.org/wiki/File:Wingdings.png
      • Wingdings 2 http://en.wikipedia.org/wiki/File:Wingdings2.png
      • Wingdings 3 http://en.wikipedia.org/wiki/File:Wingdings3.png
      • Webdings 　http://en.wikipedia.org/wiki/File:Webdings-big.png

    • 您的電腦中之所以會有Wingdings 2 字型，是因為您有安裝 MS office。若是在 Linux 中，有裝 Openoffice.org 的話，應該就會有 OpenSymbol 字型，其中也有方塊打勾字。
