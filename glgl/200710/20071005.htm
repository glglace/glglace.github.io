  • 我女兒星期二、三晚上都發燒，靠著睡冰枕平安度過，昨晚老婆也發燒了，所以我得照顧一家子妻小，今天先簡短寫一下。
    　
  • 不過，還是忍不住想分享一下，九年前我曾經找到的一個來自日本的天象模擬軟體 Stella Theater Pro，國小自然有幾個天文相關單元的教學上很好用，但由於是商業軟體所以我一直都沒有推薦和研發教學活動設計，現在終於有功能類似而且畫面更炫的「自由軟體」了！真是太令人興奮了！希望下週能隆重介紹該軟體登場。
  • 隨身碟中病毒或木馬，甚至連數位相機也遭殃，是我最近常遇到的問題，以下提出幾個預防的方法，但可能只有一段時間有效。
    1. 預防隨身碟被感染
      • 有的隨身碟有「防寫入」的功能，在插入陌生電腦時記得防寫。
      • 在隨身碟的根目錄下新增一個資料夾，命名為「autorun.inf」，這樣如果有電腦企圖在您的隨身碟上企圖寫入 autorun.inf檔案時，電腦就會跳出警示「您指定名稱的檔案已經存在」，此時您就知道該臺電腦有問題了。
    2. 關閉 Windows XP的自動執行(autorun)功能
      • 開始 => 執行 => 輸入 gpedit.msc => 跳出「群組原則」視窗
        => 左邊目錄內「電腦設定」之下選「系統管理範本」
        => 右邊選項中點選「系統」
        => 找到 「關閉自動播放」
        =>  選擇「已啟動」，並選擇停用自動播放在「所有磁碟機」。
      • 如果您無法執行 gpedit.msc，請改用 regedit，跳出「登錄編輯程式」視窗後，請搜尋 NoDriveTypeAutoRun，將值改成255 ，並重新開機。
