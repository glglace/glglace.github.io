  • 後天下午在臺北有個電子白板的捐贈儀式，因為擔心有變數，所以想先把感謝狀寄去。中午請假回到家後，先在網路上看黑貓宅急便的檔案便，說是下午五點以前送件，隔天就可以送到，於是就先看護小孩到四點多等老婆下班回家後再去寄。
    　
  • 但失算的是，日月潭的 7-11說三點以前才行，而且也不是網路看到的價錢，裝檔案的袋子是塑膠的而不是紙袋，我滿肚子的疑惑，後來有個店員說去魚池的7-11門市試試看，可是已經4:45了，趕緊跑步回家開車，老婆說要順便帶姐姐去看醫生，又延遲了一會兒，焦急的趕到魚池街上的 7-11，才總算與網頁上看到的流程一樣，花了88元順利寄出，時間是，呃？！４：５５，恰好是一首老歌的歌名，「愛你一萬年」的原曲。回想起來，有一次去日月潭的郵局想寄限時信，發現這裡只收平信，原來，到日月潭就是要悠閒的渡假呀！慢慢來就好了．．．誰叫我要住這兒呢？

  • 96下英四-week-02-20080218
    • 教材-- 培生 New Go Superkids 4
    • 進度--
      • 課本第 iv,v 頁-- 自然發音法複習 10個
      • 課本第 vi 頁-- 10,20, 30 ... 100的英文
      • 課本第 vii 頁-- 顏色的英文單字十一個
    • 備課 (約3小時)
      • 自然發音法複習20個單字，從課本拍照下來，或是用書商附的ebook光碟中裁剪，發現都太費時，而且未來會因為圖片著作權問題而無法分享，轉念想從網路上找可以自由或免費使用的圖庫中的圖片，我想起之前使用自由的向量繪圖軟體 Inkscape的網站上有看過Open Clip Art Library 圖庫，發現裡面都有 public domain 著作聲明，高達一萬多張而且有搜尋介面！實際使用時發現像 cat, pet, bed 等單字很容易找到圖片，但像 vet, hop, run 就找不到了，所以仍然不是一個完美的解決方式。
      • 自然發音複習，我構想可以將課本拍下然後將圖片拿走，只剩下英文單字，上課時讓學生將圖片排到正確位置去，看哪一組接力較快 。然而這在教學準備上需要花太多時間，放棄。
      • 想起之前在臺灣數位學校網的成果發表區中，看到新竹附小葉惠貞老師的注音符號教學在電子白板的應用影片，其中她利用井字遊戲來讓兩組競賽，九格中任選一格可看到題目，完成解答者可以畫O或X。我想用這個玩法來複習10～100的數字。
      • 想到三年前設計資訊課滑鼠練習單元時，曾經找到過一個泡溫泉的COCOA遊戲，必須在時間限制內用滑鼠依序點取數字，很適合用來複習數字的英文。
    • 上課
      1. 排定分組座位表。
      2. 說明電子白板的運作原理、保護和清潔的注意事項。為了讓每個學生有機會上臺寫一寫，讓學生輪流上臺在準備好的20個單字圖片下面寫上英文單字。
      3. 複習10～100的英文，然後四組作單淘汰賽三回井字遊戲，掀開一格看到一數字，說出正確的英文。
      4. 泡溫泉的COCOA遊戲，要求學生一面玩的時候要一面念出數字的英文，學生們都很投入。

        　
        　
    • 教學反思
      1. 當一個學生上臺作答時，其他學生顯得事不關己，並沒有把握機會學習，該如何改善？
      2. 泡溫泉的COCOA遊戲，並沒有完全複習到10～100的英語。
      3. 一節40分鐘的課，備課竟然花了三小時，沒有時間效益，希望熟練之後能縮短到一小時之內。
