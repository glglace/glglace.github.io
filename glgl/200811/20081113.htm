  • 再來是在名間社大 社大水裡分校烘焙DIY班徐老師不幸車禍往生...網頁中，看到集集分局 肇事致人於死後逃逸的連結，覺得這個警方的版本可信度最高，明天經過臺16線18公里50公尺處，再仔細看看。
  • 仔細想想，「新聞採訪」這個教學活動的發想，除了唐人亞太電視臺新聞、美國 CNN電視臺的 Student News之外，另外還有一個源頭：在準備 11/6國中綜合輔導團部落格建置的研習內容時，有一個段落是講我在別的老師的部落格上看到了什麼，其中一個是郭玉慧的讀報教育之旅，我特別喜歡其中一篇【義工媽媽週記之五】大千世界妙事多，不知不覺就使用了她所提到新聞的基本句型：「什麼人，在什麼時間，什麼地點，做什麼事」。
    　
  • 今天到學校，有兩組學生來跟我說準備好要拍了，比我預期的還要早，不過我想也好，這樣星期一六年級英文課的時候，我可以用5分鐘左右給學生們看這兩組的採訪影片，說明一下注意要點，對後續的拍攝品質會有提昇的效果，這個班級有21個學生，分為10組。
    　
  • 從第一組學生的採訪中發現需要事先提醒其他組的有：
    1. 拍攝地點的背景不能太亮，人臉會黑掉
    2. 採訪一開始要先問好
    3. 採訪者不必把受放者所講的話記下來，這樣會拖延錄影時間
    4. 攝影者的手肘要找支撐點，盡量不要晃動
    5. 採訪者講完問題之後，鏡頭焦點可以移到受訪者身上，攝影者一面看攝影機的畫面，一面隨時看看受訪者是否有任何肢體動作需要拍攝
    6. 受訪者回答完最後一個問題後，採訪者要說：謝謝您今天接受我們的採訪。


      　
  • 從第二組學生的採訪中發現需要事先提醒其他組的有：
    1. 攝影者要注意是否採訪者和受訪者的音量不夠大，背景雜音是否造成干擾，必要時則更換地點。
    2. 如果受訪者的回答比較簡短，採訪者必需更詳細的追問下去，比如說，老師您小時候放學後都做些什麼？如果老師回答：「出去玩」，那麼採訪者要追問「去哪裡玩？跟誰玩？玩什麼？」
    3. 為了減少剪接上的困難，攝影者發現有需要修正的時候，要立刻停止攝影，溝通好之後再繼續拍攝。
    4. 採訪者要事先演練，自然而大方的問出問題。


      　
  • 這兩組學生，都只花了五分鐘以內就完成了拍攝工作。後製剪接的部份，我打算用 windows movie maker，此外想摸索看看在 Ubuntu 中用自由軟體 Kdenlive 來做影片編輯。
