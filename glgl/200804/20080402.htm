將聲音與照片整合成影片

  1. 請到南投縣96年度閩南語兒童繪本創作競賽得獎作品，每人下載一本繪本的繪本檔和聲音檔。
    　
  2. 繪本檔的處理
    • 使用Power Point 去開啟繪本檔，將之另存為一張一張的圖片檔
      • [檔案] => [另存新檔] 
        • 「檔案類型」 請選擇 JPEG 檔案交換格式
        • 然後選擇匯出 「每一張投影片」
  3. 聲音檔的處理
    • 使用 Audacity 去開啟聲音檔，然後設法使之更好聽
      • 將音量調整到適當大小 (標準化)
        • [編輯] => [選取] => [全選]
        • [效果] => [標準化] => [確定]
      • 去除前後過長的多餘片段
        • [編輯] => [刪除] 
      • 去除錄音機按鈕所產生的雜音
        • [編輯] => [靜音] 
          　
  4. 使用 Windows Movie Maker 圖片和聲音整合成影片
    • [開始] => [程式集] =>  Windows Movie Maker 
      　
    • 將繪本圖片和聲音片加入到 [集合] 窗格中
      • 左邊「電影工作」選單中，「1.擷取視訊」裡面有個「匯入圖片」，點選後，到你的剛剛放繪本圖片的那個資料夾中匯入。
      • 左邊「電影工作」選單中，「1.擷取視訊」裡面有個「匯入音訊或音樂」，點選後，匯入繪本的聲音檔。
        　
    • 將繪本圖片依序拖曳到下方的腳本區中
      　
    • 按一下 [顯示時間表] 按鈕，將繪本的聲音檔拖曳到音訊/音樂軌上
      　
    • 配合聲音檔調整每一頁的時間長度
      　
    • 視情況加入轉換效果
      　
    • 製作片頭字幕
      • 左邊「電影工作」選單中，「2.編輯電影」裡面有個「建立字幕或參與名單」，點選後，選擇「在電影開頭處加上字幕」。
        • 南投縣96年度

          低年級組第一名

    • 儲存專案
      • 在還沒有製作完畢時，可以先儲存「專案」，這樣下次可以繼續修改。
      • 「專案」包含視訊、視訊轉換、視訊效果、背景音樂、配音、字幕等項目的排列及時間資訊。
        　
    • 儲存電影檔案 (網路播放用)
      • [檔案] => [儲存電影檔案] 
      • 儲存在我的電腦 => [下一步]
      • 輸入影片名稱、選擇儲存目錄  => [下一步]
      • 顯示更多選擇 => 其他設定 => 寬頻的視訊 (512Kbps)  => [下一步]
      • 開始轉換成電影檔，請稍候。
      • 完成後可觀看是否滿意。
        　
    • 儲存電影檔案 (製作DVD用)
      • [檔案] => [儲存電影檔案] 
      • 儲存在我的電腦 => [下一步]
      • 輸入影片名稱、選擇儲存目錄  => [下一步]
      • 顯示更多選擇 => 其他設定 => DV-AVI (NTSC)  => [下一步]
      • 開始轉換成電影檔，請稍候。
      • 完成後可觀看是否滿意。
