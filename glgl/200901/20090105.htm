  • 什麼是 Wiimote？
    • Wii 是任天堂公司所推出的家用遊戲主機，於2006年11月19日在美國首次發售，詳情請見中文維基百科--Wii條目
    • Wiimote 是 Wii Remote 的暱稱，即 Wii的遙控器（Remote Control）。外型像電視遙控一樣為棒狀，可以單手操作。主要的兩項功能為：指向定位及動作感應。詳情請見中文維基百科--Wiimote條目
    • 想看看 Wiimote 裡面長什麼樣兒？ 請見 Wii拆解全記錄 - Engadget 中文版
    • Wiimote運用的技術原理，請參考 Wii介紹-楊國棟
      　
  • 什麼是 Wiimote 電子白板（Wiimote Interactive Whiteboard）？
    • 美國卡內基美隆大學的韓裔博士生Johnny Chung Lee，長得胖胖的戴眼鏡跟我有點像，哎！人長得胖好像連長相也會相似，呵呵！先不說這個啦，2007年12月07日他在youtube上發表了一個影片，Low-Cost Multi-touch Whiteboard using the Wiimote，展示瞭如何利用 Wiimote 和自製的紅外線筆，實作出廉價的電子白板功能，這是他的Wiimote研究計畫的一部分。
