20071217--嘰哩呱啦ACE--第281回--教學活動設計-畫一尾魚
  • 前天週六早上，到魚池吃早餐時，剛好遇上2007臺灣魚池紅茶文化季的活動，於是改變主意外帶早餐到活動會場去邊看邊吃，還蠻愜意的，看了兩三個表演後，輪到嘉和國小扯鈴隊的表演，心想也許會遇到照明主任吧！果不其然！看到好久不見的深耕夥伴真是開心。
  • 今天早上由於學校運動會預演的關係，四年級英語課無法上，我盤算下午還有一節四年級的資訊課，如果改上英語學生一定會「哀哀叫」！可是英語課一週只有一節，要是沒上到進度會落後，於是我想了個方法讓學生滿意、老師開心。
    　
  • 我先聲明，這個教學活動用電腦並非必要的，只是解決了我教學時數不足的問題。
    　
  • 四年級英語用的課本是朗文 New Go Superkids 3，本週進度是第39-42頁。下午1330到電腦教室先用書商的課文電子書Flash檔投影出來上第39-41頁，1350講解完第42頁後，示範如何在「小畫家」上畫出課本第42頁的四個圖，1400給學生一個「題目圖檔」讓學生練習完成。
    　
  • 課本第42頁的教學內容是四個動作的英語，並有漫畫圖形示意
    1. Draw a fish.　大象同學在桌面上的紙上畫魚的輪廓
    2. Color it.　　大象同學將魚塗上顏色
    3. Spell "fish".　大象同學在心裡想如何拼出魚這個英文字
    4. Write it down.大象同學在畫好的魚下面寫上魚的英文　
      　
  • 由於想讓學生在20分鐘內完成，所以事先準備好 「題目圖檔」如下：


    　
  • 然後示範一次給學生看如何完成：


    　
  • 對應的教學目標如下：
    • 圖一
      • 會使用小畫家的「鉛筆」工具
    • 圖二
      • 會使用小畫家的「選擇」工具，並將所選區域複製、貼上
      • 會使用小畫家的「填入色彩」工具
      • 會使用小畫家的「放大鏡」工具，用來修補漏洞
    • 圖三
      • 會使用小畫家的「橢圓形」工具　（不一定要使用）
    • 圖四
      • 會使用小畫家的「選擇任意範圍」工具，並將所選區域複製、貼上
      • 會設定「透明背景」
      • 會使用小畫家的所選區域的「八個控制點」來改變圖形的大小
      • 會使用小畫家的「矩形」工具
        　
  • 結果1400學生開始動手做，到1410下課時14個學生中只有5個完成，下課時間繼續做到1420時有10個完成，不得已再跟下節課的老師借時間讓剩下的四個人完成，1430終於全部完成，在時間控制上是失敗的。
    　
  • 如果時間夠用的話，應該讓學生自訂一個主題 (如 狗、鳥、貓等等)，再做一次這個作業，教學效果會更好。
