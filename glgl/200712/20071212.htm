20071212--嘰哩呱啦ACE--第278回--從DVD影片中擷取出聲音軌
  • 如果沒有教學應用的感動與決心，再多的資訊技能的研習也是枉然。
    　
  • 從前許多人認識我，是因為我可以幫忙解決很多電腦問題，但現在我的心思並不放在資訊技術上。我最想做的，也持續在做的，是發展出精采的教學活動，使得認為資訊融入教學是無用的、被誤用的、負面看法的老師，能夠看過後拍案叫絕，進而願意去思考如何採用。
  • 以下介紹僅供各位使用於無著作權爭議之DVD影片，相關法律條文請參考：中華民國 著作權法 (民國 96 年 07 月 11 日 修正)
    　
  • 今天因有需要從DVD影片中擷取出聲音軌，然後燒錄成音樂CD以便在車上聽，網路搜尋一番後並實際操作測試成功，將過程紀錄於下供有需要的人參考：
    1. 下載免費軟體 Free DVD MP3 Ripper，安裝後執行之。
    2. Source File (指定來源檔案路徑): 指定 DVD光碟中的 VIDEO_TS 資料夾中附檔名為 vob的檔案。
    3. Output File (指定輸出檔案路徑): 指定擷取出的聲音檔要儲存到哪裡，此軟體只能將聲音檔儲存為 mp3 格式。
    4. 若您只要DVD中一小段影片的聲音檔，可以拉動在軟體視窗內左邊的影片預覽區下方的手指圖示，設定要擷取的片段之起始點和結束點。
      　
  • 問題解決:
    1. 轉檔需要您的電腦中有適當的影音編碼/解碼器，如果您不能順利轉檔，建議您安裝免費軟體 K-lite Mega Codec Pack 影音編解碼工具包
    2. 假設該DVD中有20段影片，但您實際觀看VIDEO_TS 資料夾的內容時，您可能會發現只有三、四個 vob 檔，此時若要手動用Free DVD MP3 Ripper去切割出20段，可能太辛苦了！再介紹您一個免費軟體 Vob Utils，這軟體可以根據 DVD光碟中 VIDEO_TS 資料夾內的 IFO 檔來依影片章節分割VOB檔。先用此軟體將影片切割成20段，再去轉檔就會輕鬆許多。
      　
  • 待改進
    • 此軟體1.21版目前只能一次轉一個檔案，不能批次轉檔。
