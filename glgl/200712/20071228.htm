20071228--嘰哩呱啦ACE--第290回--電子白板教學應用-10
  • 為配合水里鄉丁亥年祈安清醮祭典，自96年12月26日上午5時起至96年12月31日上午5時止為期五天齋戒素食。今天有機會去興隆超市和7-11時，看了一下是否有賣葷食，結果還是有一些，我想店家也蠻為難的，畢竟商品下架就是損失，只要願意遵守五天齋戒素食的人不去買就好了。
    • 山居歲月 - 祈安清醮祭典 :: PIXNET BLOG ::
  • 話說12/20接受教育局的指派，受贈一塊電子白板，價格我想應該是 8～10萬之間。目前暫定 1/21和1/22兩天要在永興國小辦理研習，參加人數上限20人，歡迎對電子白板在教學上的應用有興趣的老師一起來參加，可能近期教育局就會有公文或公告出現，請密切注意。
    　
  • 課程內容暫定如下：
    • 1/21、1/22 早上9:00-1200  下午1:00-4:00
    • 第一天基礎培訓 (陳薇玉/何家鳳)
    • 第二天實作與測驗：( 竹教大附小葉惠貞老師 / 新竹教育大學陳惠邦教授 )
      　
  • 可能您會說，參加完研習回校後又沒有電子白板可以使用。在此先透露南投縣教育局明(97)年應該會有電子白板相關的推動計畫出來，將會徵求15個學校左右設置示範點，如果您有興趣申請，參加這個研習應該會對您寫申請計畫有幫助。
    　
  • 12/26 到臺北參加全國中小學資訊融入教學創意競賽頒獎典禮時，會場外有一個廠商，抱歉我沒記下名字，展示的電子白板是液晶螢幕，我目測大概是60吋左右吧！可以解決老師用電子白板上課時一直被投影機強光照射的問題，但是目前一個售價 40萬元。


    　
  • 我心目中理想的電子白板，應該是這樣：
    • 不需使用投影機
    • 電腦內建
    • 顯示器旁有擴音裝置
    • 電腦不開機也可播放音樂CD、USB隨身碟上的 mp3、VCD、DVD
    • 手指即可觸控
    • Internet Ready
    • Multi-touch
      　
  • 今天還有一個令人讚嘆的新發現！ 只要臺幣四千元左右就可以擁有電子白板喔！很神奇吧？由於我有用「電子白板」當關鍵字訂閱 Google 快訊，今天收到的一則訊息是，DIY 簡易 Multi-touch 螢幕 - Engadget 癮科技，有個大大用Wii的遙控器和紅外線發射裝置，實做出電子白板的功能，真是強啊！不過這位先生，您未免長得跟我有那麼一點像啊？是不是胖子都長一個樣兒？榮幸！榮幸！
