  • 話說昨天去菜市場買菜，結果買了一棟房子回來！是這樣的，剛到菜市場入口時老婆看到水果攤和玉米攤之間的一根電線桿上有賣屋廣告，問我說要不要去看看？我 說好，去看了之後很喜歡，昨天晚上就下斡旋，稍微殺了一點價，今天中午仲介打電話來說成交！不過，我和老婆都沒有很興奮，因為並沒有長久居住的打算，我老 婆還是嚮往田園生活，這是出社會工作以來的第十次搬家，希望一切平安順利。
  • 今天晚上收到教育部自由軟體諮詢中心的凱如寄來的email，說 EzGo8 的 beta 測試版已經出來了，趕緊去下載！如果您有興趣協 助測試，請把您發現的問題提報。
  • 我下載完畢後燒成光碟，用光碟開機進入EzGo8 Beta，感覺有點慢，我習慣安裝到硬碟玩，速度會快很多。
  • 7/5 晚上 9:12 開始安裝
  • 看到「向前」、「向後」有點愣住了，大概是英文的 forward 和 backward 的翻譯吧！如果翻譯成「下 一步」、「上一步」應該會比較好懂。
  • 到這邊都是用預設的選項就好了。
  • 因為我光碟開機後有開電腦(檔案總管)來看看，所以有掛載之前我電腦裡 EzGo7的分割區，所以才會出現此畫面。
  • 因為我電腦上有 Win XP，所以我使用手動指定分割區的方式。
  • 這是我電腦原有的分割區。
  • 我打算把 EzGo7 整個覆蓋掉，所以就只有動到 /dev/sda2
  • 做好設定
  • 設定一個自己的帳密， EzGo8的預設帳號 ezgo 密碼 moe
  • 快結束了！
  • 就在這裡我出差錯 了，我沒選擇「進階」！
  • 7/5 晚上 9:22 結束安裝前設定，開始安裝
  • 奇怪的是進度怎麼會超過 100% ?
  • 7/5 晚上 9:54 安裝到硬碟完成！安裝過程花了42分鐘。
  • 重開機後只花了30 秒就進入 EzGo8 beat了！
  • 我用 來測試的電腦是五年前的，Celeron 3.2G、512 RAM。
  • 喔喔~ 偵測到我的硬碟快掛了，真的假的？
  • 系統時間也不對
  • 開 firefox 來看，有一些簡體字沒有繁體中文化。
  • Stellarium 0.10.2 跑起來很順暢，直接就都是中文介面，讚！
  • writer 的「顯示比例」工具，視窗跳出來怎麼都是英文？
  • 稍微玩了一下之後，想回Windows 所以重開機，發現 grub 的多重開機選項裡面，怎麼沒有 Windows？
  • 查了一些資料，應該是我再安裝前設定的第七個步驟沒有去「進階」裡面設定。因此我必須要去修改 grub 的開機選單，在 /boot/grub/grub.cfg。
  • 按右鍵，以系統管理員的身份開啟，然後 找 grub.cfg 中最後一段文字：
    • ### BEGIN /etc/grub.d/40_custom ### 開頭
    • ### END /etc/grub.d/40_custom ### 結尾
    • 先將開頭和結尾中間的文字刪除掉，然後加入以下幾句
      menuentry "進入WINDOWS作業系統" {
            insmod ntfs
            set root=(hd0,1)
            chainloader +1
            }
    • 存檔，重開機，搞定！

  • 您會看到兩行跟 windows 有關的選項，是因為我想試試看中文字能不能出現而已。
