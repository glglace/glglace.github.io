  2. 如何一口氣縮小一大堆數位照片？
    • 自己儲存的需求分析
      • 使用數位相機拍照，設定越高的畫素當然就能把細節拍得越好，但是如果沒有沖洗成實體照片的需求，高解析度的照片反而會造成儲存的困擾。
      • 以我個人而言，我只儲存 2400*1800 畫素，大約是將近五百萬畫素。
      • 以2008年為例，我儲存了5000張照片，平均一張數位照片 400KB，佔用了硬碟空間 2G。
      • 如果以千萬畫素等級的數位相機為例，平均一張數位照片 2M來算，佔用 10G的硬碟，若要燒光碟儲存也是一個困擾。
    • 給別人看的需求分析
      • 若有需要轉寄給別人、製作網頁及插入的檔案中的時候，也許就會希望檔案小一點比較好。
      • 如果您不知道到底要把照片縮小到什麼程度，請參考以下數值：
        •  email 寄給別人：縮小到 1024*768畫素
        • 做網頁插圖(大)：縮小到 800 * 600畫素
        • 做網頁插圖(中)：縮小到 640 * 480畫素
        • 做網頁插圖(小)：縮小到 320 * 240畫素
        • 做簡報，一張投影片中一照片：縮小到 1024*768畫素
        • 做簡報，一張投影片中六照片：縮小到 400*300畫素
        • 做A4檔案，一頁兩張照片：縮小到 1280*960畫素
        • 做A4檔案，一頁四張照片：縮小到 800*600畫素
        • 做A4檔案，一頁六張照片：縮小到 640*480畫素
        • 以上都是講橫式照片，若是直式照片請將數據中長寬對調
    • 超方便的免費工具
      • Microsoft Windows XP 的 PowerToys 裡面的 image resizer，安裝好了之後，在檔案總管中只要選取一張或多張照片，然後按右鍵，選擇 Resize Pictures，會有四種預設的大小可以選擇，如果都沒有中意的，可以按 Advanced 按鈕自訂影像大小，如此就可以輕鬆的轉換影像大小。





        如果您的照片分散在不同的資料夾中，建議使用作業系統的「搜尋」功能，並指定大小，最少是1000KB，然後在搜尋的結果中一次縮小之，注意！縮小的過程中會耗用系統大量資源，所以不要再操作電腦，去休息一下吧！


        　
      • 上述小工具很好用，可惜不知為何，有的電腦在安裝之後，並無法使用？



        　
  3. 如何「縫合」幾張數位照片成一個大圖？
    • 需求分析
      • 拍風景的時候，您會不會覺得眼睛看到的和相機拍出來的差很多嗎？這是因為人眼的視野範圍大約是 135*200度，但是一般相機的視野範圍則只有 35*50度而已。
      • 在教室內拍照片時，有時會因為拍攝者與被拍攝者之間的距離過近（再後退就撞牆了呀！）以致於拍出來的照片無法達到拍攝者心中想要呈現的樣貌。
      • 為了克服這個落差，我們可以對著一個風景拍多張照片，照片和照片之間的取景範圍有部分是重疊的，然後再將之拼貼在一起，就可以呈現較大的視野。
        　
    • 神奇的數位相片2D縫合軟體 (傳統的縫合技術只能做到水平方向)
      • 官方網站：http://www.cs.ubc.ca/~mbrown/autostitch/autostitch.html
      • 軟體作者：帥哥 Matthew Brown是個加拿大的博士後研究學者。
      • 使用授權：Free Demo Version 免費展示版
      • 軟體特色：
        • 一般縫合軟體只能沿著水平方向縫合，有時縫得不好還要使用者自己調整，Autostitch 特別之處在於，水平垂直方向都可以縫合，而且使用者什麼事都不用做！
      • 注意事項：
        • 相鄰照片必須至少有三分之一以上重疊。
        • 若相機有固定光圈的功能最好使用之，以達最佳效果。
      • 操作說明：
        1. 把要縫合的照片放在同一個資料夾，
          注意！目錄和檔案名稱都不能有中文。


          　
        2. 開啟 autostitch 軟體、選取照片


          　
        3. 喝杯茶，(對！您沒看錯，等它做完就好了！)


          　
        4. 驗收成果！照片存放在剛剛您所選的照片同一資料夾。


          　
      • 進階說明：
        • 在 Edit => Options 裡面有許多參數可以調整，目前我只有調整過縫合後的照片寬度大小這一項而已，其他的都沒試過。
