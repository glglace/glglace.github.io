  • 10/13 08:20 電腦公司來裝機，原本是說10/15週五要來，因怕來不及提前來。當天早上第一節課剛好是六年級的資訊課，想說這是一個很好的機會教育，讓他們參與也很 好。
  • 09:00 學生協助把主機搬到班級教室和辦公室，剩下的先暫時放在白板下方，並且協助把每一臺學生電腦的喇叭、鍵盤、滑鼠、線材，分別用一個紙箱裝起來，日後要取用 比較方便。


  • 學生在清空電腦桌上原有的設備時，電腦公司的兩位先生把新電腦的主機和螢幕拆箱。

  • 學生知道有新電腦可以用，都很興奮，做起事來也特別勤快，電腦桌擦得一乾二淨、清理與分類垃圾手腳俐落，做完交辦事 項立刻主動來問還可以幫忙做什麼，把新電腦的設備一組一組擺設好。


  • 09:28 把每臺主機用噴漆噴上校名，學生幫忙把線材插好，電腦公司李先生再整線就快多了。

  • 液晶螢幕放地上灰塵多，用三年半前的套子套起來，下週再來規劃要替換掉哪些螢幕。
  • 10:21 所有學生機裝設完畢，學生開機測試是否能用、是否有聲音。2007年是由 Windows 98 更新到 Windows XP，2010年這次則是更新到 Windows 7，等驗收過後，我再來裝上 EzGo 8。
