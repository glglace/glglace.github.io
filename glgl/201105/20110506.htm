20110506--嘰哩呱啦ACE--第960回--自由的風-17-LXDE桌面環境
  • 4/11家門口來了一隻流浪貓，三種毛色的小花貓，估計六個月大的樣子，很不怕生，看來很健康，有點想養。餵她吃過兩三天貓餅乾，後來因鄰居說怕社區裡有跳蚤所以做罷，但我們也沒趕她走，只是不再餵她吃東西，但這兩週她也都沒走，天天來我家鞋櫃最上層睡覺。
  • 自 從 4/8把阿剛老師打造的 TPET in Ezgo8 mac介面20110405版裝到電腦教士的一臺學生電腦硬碟上，體驗過它的流暢、美觀之後，我把家裡的電腦也裝上，用了十幾天覺得可以滿足我大部份使用 電腦的需求，可是在家裡的電腦比學校的舊，感覺上好像 ubuntu 的效能比 windows 差。
    　
  • 舉例來說，在我家裡同一臺電腦上，同樣是寫一篇網誌會開來用的軟體，用windows 開檔案總管、chrome、notepad、kompozer 跟在 ubuntu 開檔案瀏覽器、chrome、gedit、kompozer 比起來，感覺就快多了，嗯，只是感覺，沒有實際數據。
    　
  • 因此我在 4/20晚上有感而發噗：為了全面使用免費的自由軟體，我得去換更高貴的硬體？
    　
  • 噗友小斌回應我說：
    • 如果要用吃資源比較省的 Linux 桌面環境，可以試試採用LXDE桌面環境的Lubuntu。更多詳情可以參考這一篇：給老舊電腦的Ubuntu版本 - Ubuntu 正體中文 Wiki
      　
  • 看過該文之後，想到要安裝 Lubuntu後，重頭開始再把我需要用到的軟體一個一個加進去，就頭皮發麻！工程太浩大了。後然小斌說：用這個指令應該就可以裝 LXDE 套件試試看了： "sudo apt-get install lxde"
    　
  • 我想如果是在 ezgo8 中可以裝個 lxde 桌面環境來體驗看看再考慮要不要換，應該不錯，於是我上網搜尋，找到一篇很不錯的圖解文 如何在Ubuntu使用LXDE桌面，有務必去體驗看看。
    　
  • 我在ezgo8中成功裝好LXDE桌面環境，真的有感覺變快！很滿意，光是這樣就在同一臺電腦上感覺比windows 快了。
    　

  • 第一次進入 LXDE 會覺得怪怪的，下面怎麼一條黑黑的？
    　
  • 把 Docky 關掉就正常了
    　

  • 我想取消開機時啟動 Docky
    　

  • 在控制中心裏面的個人段落中，有個「始動應用程式」
    　

  • 將 Docky 移除，重新登入，疑？還是跑出來？
  • 沒辦法，只好每次進LXDE桌面環境時，手動將之關閉了。
