  • 每次去生鮮超市買東西時，我都會避過年輕小姐結帳那一道，因為他們總是無法讓我滿意的把東西適當的放在購物袋中，但剛剛去我是給一個20歲左右的小男生結帳，沒想到他卻是表現100分的優！我是說顧客買的東西裡有生食、熟時、蔬菜、餅乾、優酪乳、米各種東西的時候。
  • 在第945回和第946回中 我介紹過在 Windows 下可用 Universal USB Installer 把 Linux 作業系統放到隨身碟裡面去，可開機也可儲存設定。不過既然我已經把 ezgo8裝到硬碟上去了，為了要製作可開機Linux隨身碟，還得回到 Windows去，實在很遜！其實 ubuntu 裡面就已經內建有一個 「USB製作程式」了。
    　

  • 如上圖所示的路徑可以找到「USB製作程式」，全名應該是Ubuntu Live USB creator
    　

  • 首先要先下載好 Linux作業系統的 iso檔
  • 確認您的電腦上只插著一個隨身碟，以免做錯
  • 決定要用多少空間來存放檔案與設定檔，這裡有設定到就是 persistant
  • 若選擇「在關機時被丟棄」，那麼做出來的就是所謂的 live 隨身碟，用來網路購物時可以很放心，百毒不親喔。
