  • 連 續幾天的悶熱，雖有烏雲可是與一直下不來，今天中午總算下雨，但只下五分鐘就停了。下午3點時來了一場大雨，風大雨大，學校走廊全淹水了，雨勢應該有45 度斜角。放學後想說等雨小一點再回家，結果等到5點才去叫女兒收書包回家，她說她淋到與有點頭痛，回到家後要她趕緊去洗個熱水澡上床睡覺。到了晚上8點多 去幫她蓋被子時，發現她發燒了。
  • ezgo8 預設的中文輸入法是 gcin (Gtk Chinese INput application in X)，我在windows中是使用新注音，所以對應過來就是使用「詞音輸入法」，用了一個月之後，算是很能適應了。

    　

  • 4/9把原本是用caps lock 切換中英文，改成用左邊的shift按鍵來切換
    　
  • 但是詞音輸入法有點討厭，在某些時候某些按鍵組合會造成當機，我還搞不清楚是哪些地雷鍵的組合會引爆詞音輸入法當 機，例如在 chrome 中按到某些不明的鍵的組合， chrome 就當了。明確的說是chrome瀏覽器視窗會漸漸的變成灰色，然後就凍僵了，要按 alt+F4 將之強制關閉。更慘的時候，還得要使出 ctrl+alt+backspace 強制登出再重新登入。
    　
  • 5/2 20:46 我總算發現如何成功的搞當詞音輸入法的兩個方法了！
    1. 輸入三個字，在中文字的下面的虛線仍然在，可以選字的狀態下，按鍵盤上的 ctrl + shift
    2. 在使用詞音輸入法的狀態下，在任何能輸入中文的地方輸入幾個字，然後用滑鼠右鍵去按一下詞音輸入法的圖示，這樣 就會當掉
      　
  • 5/2 21:26 洗澡時想到，為什麼我會遇到詞音當機的原因了：我在使用詞音時，輸入幾個中文字後若想輸入英文，我有時候會按左邊 ctrl+shift 想切換成英數輸入，所以就當了。
    　
  • 5/3 噗友聿小綠幫 我測試，在他的ubuntu中，我上面所述的兩種方法都不會搞當詞音輸入法，所以是我的ubuntu之中可能什麼軟體跟詞音輸入法打架囉？
    　
  • 5/3 22:10 在ezgo8官方版本上測試詞音輸入，上述兩種當機情況都沒發生。
  • 5/4 01:02 測阿剛打包的 tpet in ezgo8 mac 0405.iso 詞音輸入會當
  • 5/4 01:50 測阿剛打包的 tpet in ezgo8 mac 0325.iso 詞音輸入也是會當
  • 5/4 08:31 測阿剛打包的 tpet in ezgo8 mac 0405.iso 但登入時改選 LXDE 桌面環境， 詞音就不會當了。
    　
  • 5/4 09:10 猜測「有可能」是 macbuntu 桌面佈景造成某些情形下詞音輸入法當掉，阿剛老師教我把使用者家目錄裏面的 .gconf .gconfd .gnome2 刪除掉試試看。我另外新建了一個使用者來測試，有成功回到 ezgo8 預設的桌面環境，而且詞音輸入也不會當了。
