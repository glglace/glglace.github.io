  • 母親節回南部，覺得南部好像已經進入夏天了，今天白天在南投則是有正式進入夏天的感覺，氣溫應該有超過30度，看來要把電風扇拿出來洗一洗了。還想到客廳的吊扇該怎麼擦呢？家裡沒有摺疊梯。
  • 5/7 晚上想要讓iPad透過電腦上的無線網卡上網，之前我是在Windows下做的，詳見我的網誌第830回，因為我家裡沒裝設無線AP。但我現在已經換到 ubuntu 了，不想為讓iPad上網就得切換到 Windows 去，那麼該怎麼做呢？
    　
  • 上網查到一篇ubuntu - 讓 iphone 透過 NB wify上網，圖解很清楚，但不是中文的，所以我重新抓圖如下：
    　

  • 對著「網路管理員」按右鍵
  • 編輯連線
  • 無線網路
  • 加入
    　

  • SSID 隨便您設定
  • 模式一定要選 Ad-hoc
  • 安全性暫時設「無」，等到成功了之後您再去改安全性
  • IPv4 方法一定要選「分享給其他電腦」
    　　
  • 測試連線看看！
    　
  • 如果看到上圖的畫面，那就太恭喜您了！輕輕鬆鬆不到五分鐘就搞定了！
  • 其實我並沒有這麼順利，而是花了三、四個鐘頭才成功的。
    　

  • 要是您的 iPad 有抓到無線網路訊號但是無法上網，像我第一次就是這樣，很苦惱，忽然想到讓 iPad 「忘記此網路設定」，重新得到一組 IP，結果就成功的讓 iPad 透過 ubuntu 無線上網啦！
    　
  • 檢視連線狀況
    　
  • 如果像上述那麼簡單，怎麼我會花了三、四個鐘頭呢？唉～總而言之，長話短說，就是 ubuntu 10.04 沒有正確的驅動我這臺一公升小電腦的無線網卡 ，型號是 RT3090，三、四個鐘頭都花在網路上尋找解決方法，測試了好多種方法都失敗到想罵髒話的山窮水盡之時，找到了一篇 HOWTO build the RaLink RT3090 driver on Ubuntu 10.04 LTS ，那時已經是我從晚上十點左右嘗試到半夜一點半了，抱著試最後一次，不行就要去睡覺了的疲憊心情，照著該文所述下載作者自製的rt3090-dkms package 並安裝，嚇！哇哩咧OOXX！竟然成功啦！發個噗紀念一下這段痛苦也痛快的經歷，然後就睡覺去了，隔天母親節假期還要長途開車回南部的。

    • ubuntu 查無線網卡型號的指令為 lspci | grep Wireless
