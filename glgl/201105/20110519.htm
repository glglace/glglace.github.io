  • 在windows中開啟瀏覽器，想要選取一個網頁的網址時，Chrome 和 Firefox 都一樣，只要在網址上用滑鼠左鍵點一下，就可以選取整個網址，這樣要複製貼上，或是新輸入另一個網址時很方便。
    　
  • 但是在 ezgo8/ubuntu 10.04 中就不太一樣：
    • Firefox - 網址點兩下才是全部選取
    • Chrome - 網址點三下才是全部選取
      　
  • 如果想要點一下網址就能全部選取，在ubuntu中需要設定一下：
    • Firefox
      1. 在網址列上輸入 about:config
      2. 在「篩選條件」的地方輸入：browser.urlbar.clickSelectsAll
      3. 點兩下就會把「值」從 false 變成 true
    • Chrome
      • 按鍵盤上的 Ctrl+L 或 Alt+D 就可以將網址全選，但無法再接著按滑鼠中鍵貼上？
      • 在這裡看 到很多人為此論戰，但似乎沒有結論？
      • 算了！就按滑鼠三下吧！不怕改變，就怕不知道怎麼做。
