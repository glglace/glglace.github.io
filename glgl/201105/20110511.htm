  • 今天早上上班出門前，兒子賴床不肯起床，老婆說弟弟乖，今天有「媽媽梯」抱抱下樓，比坐電梯還高級喔！女兒說那麼好！那我也要「把拔梯下樓」，我聽了覺得好玩，就說：好哇！把拔「踢」你下樓去～
  • 4/30晚上9點看到阿剛老師噗說：前二天校外教學全紀錄，使用ezgo8+imagination3製作的。感覺效果很讚！
    　
  • imagination 是一個將一堆照片後製成影片的自由軟體。就我所知：
    • 只有 Linux 版，沒有 Windows版
    • 可以在照片之間加入轉場特效
    • 可以在照片上加字幕
    • 可以加上背景音樂
    • 可以存成 VOB/OGV/FLV/3GP等影音格式。
      　
  • 5/3 下班時，畢業班導師問我說，有沒有可能教他班上學生將畢業旅行的照片製作成一個簡報秀，然後放在學生的大型液晶電視上輪播？會不會很困難？我原本是想說用 自由的簡報軟體impress在ubuntu上做，忽然想到阿剛老師的影片，於是我跟導師說沒問題交給我，我讓他們做成影片，以後還可以燒成DVD，還可 以加創用CC的配樂喔！
    　

  • 5/5 下午我有六年級的課，我規劃讓他們開 ezgo8 來把照片變影片~
    1. 下載畢旅照片並解壓縮放在桌面上
    2. 開啟Imagination軟體
      • ezgo8 主選單=>影音=>影片編輯-imagination
    3. 匯入圖片
      • 挑選十張照片來匯入
    4. 場景設定
      • 轉場效果類型，先用亂數取得，然後再自己調整
      • imagination 的官網上可以看到特效的展示
      • 設定每張照片持續五秒
    5. 字幕設定
      • 文字可以多行
      • 注意文字不要放得太接近上下左右的邊界
      • 字型大小要20以上
      • 調整文字的顏色和放置的位置
    6. 儲存影片秀
      • 存在桌面上
    7. 請老師試看，然後做修改
    8. 先做好的人可去 jamendo 試聽配樂，喜歡的記得加入書籤
      • chome 瀏覽器的網址列右邊有一顆星星，按一下就加到書籤裡
 　
  • 5/11 再讓學生做一節課
    1. 完成十張照片匯入、轉場、文字
    2. 去 jamendo 挑選喜歡的配樂
    3. 匯入mp3之前要做的事
      • 關閉 imagination軟體
      • 主選單 =>附屬應用程式=>終端機
      • 輸入  sudo apt-get install sox libsox-*
      • 開啟 imagination軟體
    4. 匯入音樂
    5. 在片尾加上配樂作者、來源網址，以及創用CC授權說明
