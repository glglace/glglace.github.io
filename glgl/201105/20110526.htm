  • 今天上了五節課又要幫忙一些行政事務，最後一節好不容易有空檔可以喘口氣，沒想到來了位先生跑到電腦教室裡面來，一開始我還以為他是來送電腦課本樣書了，後來才搞清楚他是圖書推銷員，因為我實在很累了，所以只好跟他說現在沒空而且我要去上廁所，尿遁去了。各行各業都很辛苦，大家彼此體諒一下吧！
  • 換到 ezgo8/ubuntu10.04 後，一直都不能習慣鍵盤右邊的九宮格數字鍵預設是不能使用，必須要先按一下 Num Lock 才能使用，到底是甚麼時候有需要把它關掉？實在搞不懂？如果沒人想要關掉，那為什麼預設是關閉的呢？
    　
  • 上網找資料，在手把手玩 Ubuntu網誌中找到一篇系 統啟動時自動打開 Num Lock，照著做一次後重開機，發現使用者要登入時，Num Lock 還是暗的，但登入後就亮了，這樣還是不方便！
    • 雖然沒能馬上解決我遇到的問題，但我還是很感謝有人願意分享可能的解決方法。
      　
  • 另外又找了一篇，他客電腦網路科技部落-Ubuntu開機自動開啟數字鍵盤，照著做一次，呵呵！開機後Num Lock會亮了！
    1. 主選單=>附屬應用程式=>終端機
    2. sudo apt-get install numlockx
    3. sudo gedit /etc/gdm/Init/Default
    4. 在gedit開啟的Default檔中，捲動軸拉到最下面，在 「exit 0」這一行上方插入
      • if [ -x /usr/bin/numlockx ]; then
        numlockx on
        fi
    5. 重開機
