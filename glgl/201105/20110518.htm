20110518--嘰哩呱啦ACE--第968回--自由的風-25-滑鼠中鍵
  • 五月初早上到學校上班時，在停車場停好車，經過學校廚房時，都會聞到一陣好聞的玉蘭花香。在學校生態池旁有一棵兩層樓高的玉蘭樹，開花時節感覺聞一聞一整天的精神都飽滿了。
  • 這陣子從 windows 作業系統換到 linux 作業系統，為達到相同目的的軟體操作方式有些許不同，用一陣子漸漸能適應。而有的則是在 windows 上我沒有嘗試使用過的功能，但實在是蠻好用的。
    　
  • 本回來談談滑鼠中鍵的使用。
    　
  • 在瀏覽器裡按滑鼠中鍵可以開新分頁
    • 3/24我發了一個噗說： 開始用 firefox 4 了，最不習慣的是，對超連結按右鍵，選擇新分頁或新視窗的兩個選項，順序與 firefox 3.x 是顛倒的，這能自己改成自己喜歡的順序嗎？既然是自由軟體...
    • 噗友聿小綠告 訴我：說開新分頁我習慣直接按中鍵。
    • 我說：哇！原來有按中鍵的玩法！從來都不知道呢！謝謝喔！
    • 由於我覺得firefox 4效能不佳，所以我已經改用 chrome 瀏覽器了，還好按滑鼠中鍵可以開新分頁的功能在 chrome 中也是可以的。
      　
  • 在 ubuntu 中選取一段文字後就已經複製了，只要按滑鼠中鍵就可以貼上
    • 忘了是在哪裡看到的功能，還是我偶然間發現的？
    • 這是我從來沒想過的用法，我一直都是先用滑鼠左鍵選取一段文字，然後按右鍵選複製，然後再到可輸入文字的地方，按右鍵貼上，一共要五個滑鼠動作，用中鍵功能的話，只要兩個滑鼠動作就完成了，真是愉快！
      　
  • 在 ubuntu 中按住鍵盤上的super鍵(或稱為視窗鍵)不放，然後滾動滑鼠中鍵的滾輪，就可以區域性放大螢幕畫面。
    • 這個在使用投影機教學時很好用
    • 設定方法請參考 彰化縣自由軟體工作日誌--超好用的教學螢幕放大鏡
      　
  • 在 ubuntu 中使用瀏覽器時，無法按滑鼠中鍵後有自動捲動的功能？
    • Firefox 解決方法：
      • 在網址列輸入 about:config
      • 把 general.autoScroll 改成 true
    • Chrome 解決方法：
      • 安裝 SmoothScroll 擴充功能
