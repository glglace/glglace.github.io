20110513--嘰哩呱啦ACE--第965回--自由的風-22-解壓縮rar檔
  • 前天晚上用節能再煮鍋煮了綠豆湯，昨天下午把吃不完的冰冰箱，並把一部份用冰棒盒做了六隻小冰棒，讓我懷念起小時候媽媽用長條型的夾鍊帶做的綠豆冰棒。今晚吃晚餐時把冰棒盒拿到餐桌上退冰，小孩眼睛發亮，吃飯也都不用人催促，真輕鬆。
  • 小孩意猶未盡的還要做冰棒，老婆找出五六顆放了很久沒吃的柳丁來切半，讓幼稚園中班的兒子榨汁。出乎意料的是，看來醜醜癟癟的柳丁居然裏面是好的，反而是外表光鮮的裏面已經熟爛了。
  • 昨天晚上收到同事寄來的一個 rar 格式壓縮檔，我發現無法用 ubuntu的壓縮檔管理員來開啟，於是 5/12 22:28 去噗浪問人，很快的就解決我的問題了。
    　
  • 雖然有噗友建議 Linux 下的 unrar-free 和 PeaZip 的兩種解決方案，可是阿剛老師說若當初那個壓縮檔是在windows下壓出來的，而且檔名有中文的話會變亂碼，所以我採取用wine開啟7-zip來解開rar壓縮檔。
    　
  • 我在ubuntu 10.04試驗成功的方法如下：
    1. 到 7-zip 官網下載安裝檔，放在 ubuntu 桌面上
    2. 用 wine 來安裝 7z920.exe
    3. 找個rar檔案,按右鍵,選[屬性]-[以此開啟]-[加入],[使用自訂指令], 然後打 wine ~/.wine/dosdevice/c:/Program Files/7-zip/7z.exe
    4. 如此一來，以後都只要對著 rar 壓縮檔按兩下，就會跳出 7-zip的解壓縮視窗，再指定要解壓縮到哪裡就好了。
