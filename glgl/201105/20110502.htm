  • 以下分享我是怎麼做的？

  • 這個命令稿需要用到 exiftool 套件，如果你沒有的話，請開終端機視窗並輸入下列指令安裝之：
    sudo apt-get install libimage-exiftool-perl
    　
  • 然後按照下面步驟做一次：
    1. 開啟終端機視窗
    2. 切換目前所在的目錄
      cd /home/目前使用者名稱/.gnome2/nautilus-scri­pts/
    3. 創建一個叫做 「依日期分資料夾」 的命令稿
      sudo gedit 依日期分資料夾
    4. 請輸入以下內容：
      #! /bin/sh
      while [ $# -gt 0 ]; do
      picture=$1
      f_date=`exiftool "$picture" -CreateDate -d %Y%m%d | cut -d ':' -f 2 | sed -e 's/^[ \t]*//;s/[ \t]*$//'`
      if [ ! -d $f_date ]; then
      mkdir "$f_date"
      fi
      mv "$picture" "$f_date"
      shift
      done
    5. 將「依日期分資料夾」命令稿存檔
    6. 回到檔案瀏覽器
    7. 選取一堆照片
    8. 按右鍵 => 命令稿 =>點選「依日期分資料夾」
